# 現場作業手順 適用差分マトリクス

## 1. 目的

本書は、清掃・設備・警備の標準手順に対し、建物用途、管理方式、契約役割、法令義務及び責任分界によって何を変更・追加確認するかを横断的に整理する。差分を理由に手順全体を複製せず、標準手順と物件別適用条件を分離する。

## 2. 適用方法

1. 対象の手順IDと版を特定する。
2. 物件の建物用途、管理方式、契約役割、法令義務及び責任分界を特定する。
3. 本書から該当差分を抽出する。
4. 追加工程、変更基準、必要権限、追加証跡を物件別手順又はチェックリストへ反映する。
5. 差分の承認者と適用開始日を記録する。

法令・行政指示、安全上の要件、契約・計画・規程、メーカー指定、標準手順、物件別運用を照合する。矛盾がある場合は作業者の推測で解消しない。

## 3. 差分の記載単位

| 項目 | 内容 |
|---|---|
| 適用条件 | どの用途、方式、役割又は法的条件で適用するか |
| 変更対象 | 開始条件、工程、判定基準、権限、完了条件等のどこを変えるか |
| 追加・変更内容 | 実施する差分 |
| 必要権限 | 誰の確認・承認が必要か |
| 追加証跡 | 追加で残す記録、写真、測定値、連絡履歴等 |
| 根拠 | 契約、規程、法令、計画又は承認記録 |

## 4. 建物用途による差分

| 用途 | 清掃 | 設備 | 警備 |
|---|---|---|---|
| オフィス | 執務・会議時間、機密書類、残業者に配慮 | 執務影響、休日作業、テナント系統を確認 | 残業者、来訪者、機密区画、休日入館を確認 |
| 商業施設 | 営業中の濡れ床・臭気・代替動線を厳格化 | 営業・テナント・冷蔵設備への停止影響を確認 | 群集、迷子、閉店確認、売場・搬入口を重点化 |
| ホテル | 客室稼働、静音、忘れ物、24時間利用に配慮 | 給湯・空調・昇降機等の宿泊影響を確認 | 宿泊者のプライバシー、深夜対応、客室階入室を制限 |
| 病院・福祉 | 清潔・汚染区域、感染対策、患者動線を追加 | 医療・療養継続に関わる重要負荷と切替えを確認 | 患者保護、面会、薬品・重要区画、避難困難者を考慮 |
| 学校 | 授業・行事・児童生徒の動線、薬剤管理を追加 | 授業時間、長期休業、実験・給食設備を確認 | 登下校、児童生徒保護、保護者・外来者確認を強化 |
| 共同住宅 | 居住者の生活時間、私有物、騒音・臭気に配慮 | 断水・停電・給湯停止の住戸影響と周知を確認 | 居住者・来訪者、宅配、共用部のプライバシーを考慮 |
| 工場・物流 | 生産・食品・危険区域との交差汚染を防止 | 生産停止、安全設備、危険物、ロックアウトを優先 | 車両、搬入口、危険区域、在庫・情報資産を重点化 |
| データセンター | 防塵、帯電、持込品、二人体制等を追加 | 冗長系、変更管理、連続稼働、入室権限を厳格化 | 多要素認証、共連れ、持込・持出、入退室証跡を強化 |

## 5. 管理方式による差分

| 管理方式 | 開始・引継ぎ | 実施中 | 異常時 | 完了・証跡 |
|---|---|---|---|---|
| 常駐管理 | 前勤務者、日誌、中央監視、当日工事から引継ぐ | 現地責任者と随時連携 | 即時現場確認、応援、関係者調整 | 次勤務者へ対面又は勤務記録で引き継ぐ |
| 巡回管理 | 訪問前に遠隔情報、鍵、入館、携行品を確認 | 滞在時間、複数物件の優先度を管理 | 遠隔一次確認後、出動・専門業者・緊急機関を選択 | 写真、到着・退出時刻、未確認点、次回訪問を追加 |
| 遠隔監視 | センサー範囲、通信状態、現場連絡先役割を確認 | 複数情報を相関し誤報・通信断を識別 | 現場確認なしで断定せず、出動条件に従う | 受信、確認、連絡、出動判断の時刻を保存 |
| 機械警備・駆付け | 警報種別、到着目標、単独進入条件を確認 | 現場安全と侵入痕を優先 | 危険時は進入せず警察・消防・応援へ接続 | 警報受信から引渡しまでの時系列を記録 |

## 6. 契約役割による差分

| 役割 | 指示・計画 | 現場実施 | 確認・報告 | 決定してはならない事項 |
|---|---|---|---|---|
| 元請BM | 契約仕様を手順・指示へ展開し再委託を統制 | 自社実施又は再委託先を支援 | 自主検査、統合報告、顧客説明、是正管理 | オーナー権限の投資・重大リスク受容 |
| 再委託先 | 元請指示と自社安全基準を照合 | 契約・資格・教育範囲で実施 | 自主確認後、元請へ速報・完了報告 | 無承認の仕様変更、追加費用、利用再開 |
| 専門業者 | 対象設備・専門作業の条件を確認 | 専門資格・メーカー要領に従う | 技術所見、測定値、交換部品、残留リスクを提出 | 建物全体の運用継続・営業判断 |
| 施設側担当 | 入館、区域、利用者調整、運用情報を提供 | 契約上の担当範囲のみ実施 | 施設影響と利用再開条件を確認 | 専門資格が必要な技術判断 |

再委託先の「作業完了」、元請の「検収完了」、顧客の「承認完了」は別状態として記録する。

## 7. 法令義務による差分

| 確認項目 | 差分として定義する内容 |
|---|---|
| 義務主体 | 所有者、占有者、管理権原者、事業者等の誰が義務を負うか |
| 実施資格 | 有資格者、選任者、監督者、立会者の要否 |
| 実施方法・周期 | 法定方法、回数、時期、測定条件、校正要件 |
| 様式・届出 | 法定様式、行政・検査機関への報告、期限 |
| 保存 | 保存対象、保存期間、改ざん防止、閲覧権限 |
| 異常時義務 | 停止、通報、改善、再検査等の条件 |
| 契約との境界 | 法定業務と自主点検・日常作業を混同しないための区分 |

法定業務に該当する可能性がある場合、標準手順のチェック完了だけで法令適合を宣言しない。

## 8. 責任分界による差分

| 判断 | 標準的な実行者・決定者の整理 |
|---|---|
| 承認済み手順の実施 | BM又は再委託先の作業者（L0） |
| 軽微な一次対応・安全確保 | 教育・権限範囲内の現場作業者（L1） |
| 臨時運転、経路変更、部分停止 | 設備・警備・施設責任者等（L2） |
| 追加費用、契約外作業、長時間の利用制限 | PM、FM、オーナー等の契約・予算権限者（L3） |
| 更新投資、重大リスク受容、長期停止 | オーナー等の最終権限者（L4） |
| 緊急停止・退避 | 切迫危険時は現場の緊急権限。ただし即時報告する |
| 技術的復旧 | 資格・専門性・操作権限を持つ者 |
| 施設利用再開 | 施設運用上の権限者。技術的復旧と分離する |

## 9. 手順別の主要差分確認点

| 手順群 | 建物用途 | 管理方式 | 契約役割 | 法令義務 | 責任分界 |
|---|---|---|---|---|---|
| `PROC-COM-001` 引継ぎ | 重要区画・利用予定 | 対面／遠隔／訪問間 | 引継ぎ元・受領者 | 必須記録・保存 | 未解決案件の受領責任 |
| `PROC-COM-002` 安全確認 | 利用者・危険区域 | 単独・通信・応援 | 安全指示系統 | 資格・作業規制 | 開始保留・中止権限 |
| `PROC-COM-003` 区域管理 | 代替動線・営業影響 | 無人時の表示・解除 | 設置・承認・巡視 | 立入規制要件 | 閉鎖・利用再開 |
| `PROC-COM-004` 異常上申 | 重要負荷・利用者影響 | 到着・出動条件 | 速報経路 | 通報・届出 | 停止・費用・リスク受容 |
| `PROC-COM-005` 完了報告 | 顧客・利用者報告 | 遠隔証跡 | 実施・検収・承認 | 法定様式・保存 | 未完了承認 |
| `PROC-CLN-001〜006` | 衛生・営業時間・材質 | 携行品・写真・再訪 | 自主検査・検収 | 廃棄物・薬剤・衛生 | 区域閉鎖・再清掃・追加費用 |
| `PROC-EQP-001〜008` | 重要負荷・停止影響 | 遠隔監視・駆付け | 操作・専門判断範囲 | 保安規程・資格 | 停止・復旧・再開 |
| `PROC-SEC-001〜008` | 人物・群集・重要区画 | 常駐・機械警備 | 指揮・通報・引渡し | 警備・消防・個人情報 | 入館拒否・区域閉鎖・公的機関指揮 |
