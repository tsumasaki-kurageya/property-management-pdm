# PROC-EQP-006 緊急停止・隔離

## 1. 目的

切迫した人命危険又は設備・施設被害の拡大を防ぐため、権限と安全を確認できる範囲で設備・エネルギー源を緊急停止し、再作動・再加圧・流入等を防止する。停止操作、安全状態、隔離状態及び後続責任を追跡し、原因究明、修理、再起動及び利用再開と区別する。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-10-03` | 一次対応を行う | 危険停止と安全隔離を行う |
| 関連業務 | `BM-08-02`、`BM-08-07` | 起動・停止、警報対応 | 異常認知と停止操作へ接続する |
| 関連業務 | `BM-10-02` | 緊急度を判定する | 人命・被害拡大の切迫性を判断する |
| 関連業務 | `BM-12-08` | 関係者へ連絡する | 停止影響・区域・代替運用を周知する |
| 関連業務 | `BM-17-05`〜`BM-17-07` | 安全・リスク・事故管理 | 隔離、事故、証跡を管理する |

## 3. 適用範囲

- 火災、感電、漏電、破裂、高温・高圧、重大漏水、巻込等の被害拡大防止
- 承認済み緊急停止装置、遮断器、弁等による停止
- 電気、機械、流体、熱、圧力、位置エネルギー等の隔離
- ロック・タグ、区域管理、影響把握及び専門担当への引渡し

資格外の活線作業、危険区域への接近、原因不明状態での再投入・再起動、保護機能の無効化及び修理は対象外とする。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 第一認知者 | 退避、緊急通報、権限内の停止、事実伝達 |
| 停止操作者 | 対象照合、停止操作、応答確認、時刻記録 |
| 設備・保安責任者 | 系統影響、隔離方法、残留エネルギー、安全状態の判断 |
| 警備・防災担当 | 区域、避難、消防等の誘導・連携 |
| 施設運用責任者 | 利用制限、代替運用、関係者周知を決定する |
| 専門業者 | 専門隔離、診断、修理、復旧確認を行う |

## 5. 開始条件

切迫時は書面確認を待たず退避・通報を優先する。停止操作は可能な範囲で次を確認する。

- [ ] 事象、場所、対象設備・系統、危険範囲を確認した
- [ ] 停止により別の人命危険・重要負荷停止が生じないか確認した
- [ ] 操作位置へ安全に接近でき、退路・通信を確保した
- [ ] 緊急停止権限、必要資格、正しい停止点を確認した
- [ ] 消防・救急・警察、設備・施設責任者への連絡を開始した
- [ ] 停止失敗時の退避・代替遮断手段を確認した

## 6. 必要資料・資機材

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 資料 | 緊急操作票、系統図、保安規程、SDS | 最新版、停止影響 |
| 操作 | 停止装置、遮断器・弁の識別 | 対象、遠隔・現地差 |
| 隔離 | ロック、タグ、閉止板、検電・検圧器等 | 適合、校正、個人管理 |
| 安全 | PPE、検知器、消火・区域用品 | 危険への適合 |
| 通信 | 無線、電話、緊急連絡網 | 時刻同期、代替手段 |
| 記録 | 緊急対応票、写真 | 操作前後、改変管理 |

## 7. 緊急判定

| 状態 | 代表条件 | 初動 |
|---|---|---|
| 退避優先 | 火炎、アーク、ガス、倒壊、破裂等で接近不能 | 接近せず通報・区域拡大 |
| 緊急停止可能 | 正しい停止点へ安全に接近でき、停止が被害を低減 | 緊急権限で停止し即時報告 |
| 停止判断要請 | 停止による生命維持・防災等への重大影響がある | 安全を保ち指定責任者へ即時判断要請 |
| 情報不足 | 対象・系統・影響が識別不能 | 推測操作をせず退避・専門担当へ |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 退避・通報し、危険範囲と対象を確認する | 自身・第三者の安全を優先 | 接近不能なら停止操作しない | 認知、通報時刻 |
| 2 | 設備・系統・停止点を表示と資料で照合する | 誤停止を防止できる | 識別不能なら専門判断へ | 対象、照合者 |
| 3 | 停止影響と緊急権限を可能な範囲で確認する | 停止の二次影響を把握 | 切迫時は緊急権限で実施 | 判断、根拠 |
| 4 | 安全な位置から承認済み緊急停止を実行する | 対象の停止応答がある | 無応答時は反復せず退避 | 操作、時刻 |
| 5 | 監視・現地で停止状態と残る危険を確認する | 危険源低減を確認 | 不十分なら代替隔離を要請 | 表示、測定 |
| 6 | 権限・資格者がエネルギー源を隔離する | 全危険源と逆流・蓄積を考慮 | 範囲外は区域維持し引渡し | 隔離点、操作者 |
| 7 | ロック・タグ等で再作動を防止する | 誰が何をなぜ隔離したか明確 | 管理不能なら監視を配置 | ロック・タグID |
| 8 | 残留エネルギーを解放・拘束し無エネルギーを確認する | 指定方法で検証済み | 検証不能は安全成立としない | 方法、値、確認者 |
| 9 | 区域・利用制限・代替運用を設定し周知する | 停止後の影響を管理 | 重要影響は運用責任者へ | 区域、周知先 |
| 10 | 専門担当へ状態、隔離、残留危険、禁止事項を引き渡す | 受領と責任移管が明確 | 受領まで監視・区域を維持 | 受領者、時刻 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 停止済み | 停止操作の応答を確認 | 隔離・検証を継続 | 手順継続 |
| 安全隔離成立 | 危険源を隔離し再作動防止・残留確認済み | 専門担当へ引渡す | 条件付き可 |
| 暫定安全 | 完全隔離不能だが区域・監視・代替措置あり | 期限・担当付きで継続 | 条件付き可 |
| 安全未成立 | 危険源継続、対象不明、隔離・検証不能 | 退避・区域拡大・緊急連絡 | 不可 |

## 10. 異常時対応

- 停止装置が応答しない場合、無断で操作を反復又は分解しない。
- 複数エネルギー源、逆送、蓄圧、惰性回転、重力落下を単一停止で安全とみなさない。
- 消防・救急等が指揮する現場では指示に従い、現場変更を記録する。
- 完全隔離できない場合は、危険区域、監視、再接近禁止及び期限を明示する。
- 異常上申は`PROC-COM-004`、修理後の復旧は`PROC-EQP-007`へ接続する。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 退避、通報、第三者保護 | 全員 |
| 承認済み緊急停止 | 物件別緊急権限 |
| 電気・流体等の隔離、無エネルギー確認 | 必要資格・指定権限者 |
| ロック・タグ解除、再投入、再起動 | 設置者本人又は規程上の解除権限者 |
| 長期停止、代替運用、広範な利用制限 | L3以上 |
| 技術復旧・施設利用再開 | 本手順とは別の指定権限者 |

## 12. 記録・完了条件

- [ ] 認知、退避、通報、停止、隔離、確認、引渡しの時刻を記録した
- [ ] 対象設備・系統、停止点、操作前後状態、停止影響を記録した
- [ ] 全隔離点、ロック・タグID、残留エネルギー確認を記録した
- [ ] 危険区域、監視、利用制限、代替運用、周知を記録した
- [ ] 暫定措置、残留危険、禁止事項、担当・期限を記録した
- [ ] 専門担当の受領と、再起動禁止・解除権限を確認した

停止ボタンを押しただけでは完了しない。安全隔離又は暫定安全を成立させ、後続責任が受領されていることを完了条件とする。原因究明、修理、再起動、技術的復旧及び施設利用再開は別状態である。
