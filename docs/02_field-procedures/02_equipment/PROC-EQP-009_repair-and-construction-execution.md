# PROC-EQP-009 修繕・工事実施

## 0. 文書管理

| 項目 | 内容 |
|---|---|
| 手順ID | `PROC-EQP-009` |
| 手順名 | 修繕・工事実施 |
| 領域 | 設備 |
| 版 | 0.1 |
| 状態 | 初版草案 |
| 作成日 | 2026-07-24 |

> 本文書は、承認済みの修繕・工事を着工し、施工者の自主検査と施工記録をそろえて完了検査へ引き渡すまでの共通管理ゲートを定める。工種別施工要領、設計図書、メーカー手順、施工計画書、保安規程、法定手続及び資格者の専門判断を置き換えない。

## 1. 目的

承認済み範囲、施工条件、安全・品質基準及び施設運用への影響を着工前に照合し、施工中の逸脱、追加危険、周辺影響及び変更を管理する。現場で承認済み範囲を無断変更せず、工事写真、試験、自主検査、残工事及び完成図書を追跡可能にして、`PROC-EQP-007`の試運転・復旧確認へ引き渡す。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-10-09` | 修繕・工事を実施する | 承認済み修繕・工事の着工、施工、変更統制、自主検査及び引渡しを行う |
| 前工程 | `BM-10-05`〜`BM-10-08` | 方法検討、見積、顧客承認、手配 | 承認範囲、費用、工期、施工者、許可及び停止条件を受領する |
| 後工程 | `BM-10-10` | 完了検査・引渡しを行う | 施工結果、試験、残工事及び完成図書を引き渡す |
| 関連業務 | `BM-10-11` | 修繕履歴を登録する | 原因、処置、部品、変更及び写真を履歴化する |
| 関連業務 | `BM-13-01`〜`BM-13-06`、`BM-13-11` | 指示、記録、確認、承認、上申 | 指示、実績、変更、異常及び受領を記録する |
| 関連業務 | `BM-17-01`〜`BM-17-08`、`BM-17-11` | 品質、安全、リスク、事故、区域管理 | 施工中の品質・安全・周辺影響を統制する |

## 3. 適用範囲・除外

### 3.1 適用対象

- 故障・劣化した設備、配管、配線、部品、建築付帯部等の修繕・交換
- 承認済み設計、仕様、見積、工程及び施工計画に基づく小規模な改修・更新
- 専門業者が実施し、BM又は元請が作業区域・施設運用・引渡しを統制する工事
- 停止・隔離後の施工から、自主検査を経て試運転・完了検査へ渡すまで

### 3.2 対象外

- 修繕方法、設計、投資、見積、発注又は顧客承認そのもの
- 工種固有の加工、接続、組立、溶接、電気・配管等の詳細施工方法
- 承認されていない追加工事、契約外作業又は現場判断による仕様変更
- 法令上の届出、検査、資格者業務又は石綿等の専門調査を一般作業者が代替すること
- 技術的復旧、施設利用再開、契約上の検収又は修繕案件の終結判断

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 顧客・発注権限者 | 修繕範囲、予算、工期、重要な変更及び契約上の受領を承認する |
| BM・元請責任者 | 承認条件、施設影響、関係者調整、証跡受領及び後工程への接続を統制する |
| 工事責任者 | 施工計画、作業指示、資格、品質、安全、工程、変更及び自主検査を管理する |
| 作業責任者・専門業者 | 工種別施工要領に従って施工し、状態、実績、異常及び変更要求を報告する |
| 設備・保安責任者 | 停止、隔離、残留エネルギー、操作禁止、復旧条件及び保安規程適合を確認する |
| 施設運用責任者 | 停電・断水・騒音等の利用影響、代替運用、周知、区域及び利用制限を判断する |
| 監理・検査担当 | 指定された立会点、隠蔽前、材料、施工品質及び試験結果を確認する |

施工者の自主検査、技術確認者の復旧判定、BM・元請の完了確認、顧客の検収及び施設運用責任者の利用再開は別の確認として記録する。

## 5. 着工前ゲート

次の必須条件を満たさない場合は着工しない。緊急時の安全確保は`PROC-EQP-006`を優先し、事後の恒久修繕を本手順へ接続する。

### 5.1 承認・施工条件

- [ ] 修繕・工事案件、承認済み範囲、設計・仕様、数量、見積、工期及び除外を特定した
- [ ] 施工計画、工程、工種別施工要領、品質基準、試験・検査要領及び中止条件を確認した
- [ ] 作業責任者、施工者、資格者、監視者、立会者、承認者及び連絡経路を確認した
- [ ] 必要な届出、許可、資格、火気・高所・閉所等の作業許可及び行政・施設条件を確認した
- [ ] 石綿等の事前調査対象、調査結果、表示・報告及び未調査部位への接触禁止を確認した
- [ ] 材料・部品の型式、規格、数量、納入状態、証明書及び代替品禁止条件を確認した

### 5.2 停止・施設影響

- [ ] 対象設備・系統、上流・下流、重要負荷、他工事及び既存異常を特定した
- [ ] 停電、断水、空調停止、警報停止、騒音、振動、臭気、粉じん及び搬出入の影響を確認した
- [ ] 停止・隔離、ロック・タグ、残留エネルギー確認及び解除権限を確認した
- [ ] 利用者・テナント・警備・防災・関係業者への周知、代替運用及び緊急時対応を確認した
- [ ] 作業区域、資材置場、搬入経路、避難経路、火気・粉じん・漏水等の拡大防止を確認した
- [ ] 作業前状態、既存損傷、設定、配線・配管、弁・スイッチ位置等を写真・図面で記録した

## 6. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 作業指示、承認範囲、図面、仕様、施工計画及び現物を照合する | 対象・条件・除外が一致する | 不一致を解消するまで着工しない | 案件、版、承認 |
| 2 | 朝礼・危険予知を行い、役割、工程、作業競合、中止条件を共有する | 全員が担当と危険を復唱できる | 理解・資格不足者を作業させない | 参加者、資格、指示 |
| 3 | 区域、養生、搬入、消火・漏えい対策、監視及び通信を設定する | 第三者・施設への影響を管理できる | 区域・対策を再設定する | 区域、養生、周知 |
| 4 | 指定権限者が停止・隔離し、安全状態を検証して引き渡す | 再作動と残留危険を防止できる | `PROC-EQP-006`へ接続し施工を保留する | 隔離点、タグ、確認者 |
| 5 | 材料、部品、工具、測定器、施工箇所及び作業前状態を確認する | 仕様・数量・校正・既存状態が一致する | 代替使用や推測施工をしない | 材料、型式、前状態 |
| 6 | 工種別施工要領に従い、指定工程ごとに施工する | 承認範囲・手順・品質基準内 | 安全な状態で中止し責任者へ報告する | 実施者、時刻、工程 |
| 7 | 立会点・停止点・隠蔽前確認で、寸法、接続、締結、設定等を確認する | 後から確認できない品質が検証済み | 確認前に隠蔽・次工程へ進まない | 検査、写真、測定値 |
| 8 | 工程、品質、安全、周辺影響及び設備状態を継続監視する | 計画内で影響が管理されている | 作業中止、区域拡大、上申を行う | 巡視、異常、是正 |
| 9 | 現場条件、数量、材料、経路、方法又は影響の変更要求を識別する | 変更の有無と承認要否が明確 | 変更対象を施工せず8章へ進む | 変更要求、根拠 |
| 10 | 不適合・損傷・誤施工を識別し、対象を保留・表示・隔離する | 無断手直しや証跡消失がない | 不適合処理と再承認を行う | 不適合、処置、承認 |
| 11 | 施工者が自主検査、清掃、工具・残材確認及び仮設状態確認を行う | 施工計画上の完了条件を満たす | 残工事・不適合として明示する | 自主検査、残工事 |
| 12 | 施工記録、写真、試験成績、材料証明、変更、残工事及び完成図書をまとめる | 実際の施工状態を追跡できる | 不足を期限・担当付きで管理する | 完成図書一覧 |
| 13 | 工事責任者とBM・元請が引渡し条件を照合する | 試運転・完了検査を安全に開始できる | 引渡し保留又は条件付き引渡しとする | 判定、受領者、時刻 |
| 14 | `PROC-EQP-007`へ施工状態、隔離、試験条件、残留リスクを引き渡す | 後工程の責任者が受領した | 受領まで区域・隔離を維持する | 引渡し、制限、期限 |

## 7. 施工中の状態判定

| 状態 | 基準 | 標準対応 | 継続可否 |
|---|---|---|---|
| 計画内 | 承認範囲、施工条件、品質・安全基準及び工程内 | 記録して施工継続 | 可 |
| 要確認 | 軽微な不明、関連作業の遅れ、周辺影響の変化等があるが安全状態を維持できる | 対象工程を保留し責任者確認 | 対象外のみ可 |
| 変更承認待ち | 仕様、材料、数量、経路、工法、停止影響、工期又は費用に変更が必要 | 変更範囲を施工せず再承認 | 不可 |
| 不適合 | 承認仕様・品質基準から逸脱、誤施工又は損傷がある | 識別・隔離し不適合処理 | 不可 |
| 危険 | 人命、火災、感電、漏えい、倒壊、設備重大損傷又は第三者被害のおそれ | 全員が中止・退避・通報可能 | 不可 |

## 8. 変更・中止・再承認

### 8.1 現場裁量で実施できる範囲

- 承認済み施工計画内での安全な作業順、要員配置及び養生の軽微な調整
- 承認済み範囲内の工具・仮設方法の調整で、品質、停止影響、工期及び費用を変えないもの
- 人命保護・被害拡大防止のための作業中止、退避、区域拡大及び緊急連絡

### 8.2 中止して再承認する条件

- 設計、仕様、材料・型式、数量、施工範囲、経路又は工法を変更する
- 停電・断水等の停止範囲、期間、重要負荷、避難・防災又は利用者影響が変わる
- 追加費用、契約外作業、工期変更、別業者手配又は保証条件への影響が生じる
- 未調査部位・未知物質、石綿等の疑い、図面と異なる隠蔽状態又は新たな危険を発見する
- 品質基準を満たさず、承認済みの是正方法がない

変更要求には、発見事実、変更理由、代替案、品質・安全・施設影響、費用・工期、図面・写真及び復旧条件を添える。口頭指示を受けた場合も、緊急安全措置を除き、所定の承認記録が成立するまで変更施工を開始しない。

## 9. 権限・承認境界

| 判断・行為 | 標準権限 |
|---|---|
| 承認済み計画・施工要領どおりの施工 | 作業責任者・必要資格を持つ施工者（L0） |
| 計画内の軽微な順序・要員・養生調整 | 工事責任者（L1） |
| 安全のための中止・退避・通報 | 全員 |
| 停止・隔離の実施、変更及び解除 | 設備・保安規程上の指定権限者 |
| 仕様、材料、数量、施工方法、停止影響又は工程の変更 | BM・元請・設計・顧客等の所定承認者（L2〜L3） |
| 追加費用、契約外作業、工期変更 | 発注・契約権限者（L3以上） |
| 更新、長期停止又は重大リスク受容 | 投資・施設運用権限者（L4） |
| 自主検査合格 | 工事責任者又は施工者の品質責任者 |
| 技術的復旧判定 | `PROC-EQP-007`で定める技術確認者 |
| 区域解除・施設利用再開 | 施設運用権限者 |
| 契約上の検収・案件終結 | 顧客・元請等の契約権限者 |

## 10. 完了前確認・引渡し

### 10.1 施工状態

- [ ] 承認範囲の施工、自主検査及び指定立会を完了した
- [ ] 接続、締結、固定、絶縁、漏えい、清掃、異物・工具・残材を確認した
- [ ] 仮設、保護、バイパス、接地、弁・スイッチ、ロック・タグの状態を明示した
- [ ] 不適合、残工事、未試験、未提出書類及び条件付き事項を一覧化した

### 10.2 記録・完成図書

- [ ] 作業者、資格、開始・終了時刻、工程、材料・部品、数量及び施工箇所を記録した
- [ ] 作業前、施工中、隠蔽前、完成及び異常・変更の写真を識別可能に整理した
- [ ] 測定値、試験成績、材料証明、廃棄・搬出、届出・許可及び立会記録を整理した
- [ ] 設計・仕様との差、変更承認、実施工図、設定・構成変更及び保証情報を整理した
- [ ] 残留リスク、利用制限、監視条件、再停止条件、担当及び期限を記録した

## 11. 完了条件

本手順は、承認済み範囲の施工と施工者の自主検査が完了し、施工記録、工事写真、試験成績、変更承認、残工事及び完成図書の状態が明確で、工事責任者とBM・元請が`PROC-EQP-007`の開始可否を判定し、後工程責任者が受領した時点で完了できる。

次の状態は本手順の完了と同義ではない。

1. 設備の技術的復旧
2. 作業区域の解除
3. 施設・テナントの利用再開
4. 顧客又は元請による契約上の検収
5. 修繕履歴・台帳・図面の最終更新
6. 修繕案件の終結

残工事又は書類未提出があっても、安全と後工程開始条件を損なわず、内容、制限、担当、期限及び承認者が明確な場合は条件付きで引き渡せる。安全・品質・試運転条件に影響する未完了がある場合は引渡し保留とする。

| 責任分界 | 設計・仕様変更、停止、追加費用、残留リスク、区域解除、利用再開及び検収の承認者 |
