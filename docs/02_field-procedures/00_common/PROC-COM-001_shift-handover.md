## 1. 目的

勤務交代又は担当変更時に、業務を継続するために必要な状態、情報、物品、権限及び未解決事項を引継ぎ者と受領者が照合し、責任移管の成立を明確にする。説明しただけの状態と受領が完了した状態を分け、業務の空白、二重対応、対応漏れ及び情報欠落を防ぐ。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-05-10` | 勤務・担当を引き継ぐ | 受領可否と責任移管を確定する |
| 関連業務 | `BM-05-04` | シフトを作成する | 予定勤務・担当と実態を照合する |
| 関連業務 | `BM-13-01`〜`BM-13-04` | 作業開始・内容・完了を記録・報告する | 前勤務の結果と新勤務の開始を記録する |
| 関連業務 | `BM-17-06` | 危険予知・リスク評価を行う | 継続中の危険と対応条件を確認する |

## 3. 適用範囲

### 3.1 適用対象

- 勤務帯、シフト又は当番の交代
- 作業途中、障害対応中又は継続監視中の担当変更
- 清掃、設備、警備及び複数領域間での継続案件の受渡し
- 対面、遠隔又は無人時間帯を挟む引継ぎ
- 元請け、再委託先、施設担当等の契約境界をまたぐ担当移管

### 3.2 適用除外・使用禁止条件

- 緊急事象に対する人命保護、退避又は緊急通報を遅らせる形式的な引継ぎ
- 資格、教育、配置又は契約上の受領条件を満たさない者への移管
- 重要情報、物品又は権限の不一致を未報告のまま受領済みとすること
- 口頭説明だけで記録・現物・対象状態を確認せず責任移管を成立させること
- 個人情報、鍵情報、設備情報等を受領権限のない相手又は媒体へ伝達すること

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 引継ぎ者 | 事実、実施結果、未解決事項、対象状態及び受渡物を整理して説明する |
| 受領者 | 記録・現物・状態を照合し、復唱又は再確認して受領可否を示す |
| 現場責任者 | 条件付き受領、開始保留、担当変更及び応援配置を判断する |
| 上位責任者・顧客側権限者 | 契約外対応、利用制限、業務縮小、リスク受容等を決定する |
| 専門担当 | 設備、警備、防災、衛生等の専門状態と操作権限を確認する |

引継ぎ者と受領者を同一人物として記録しない。遠隔又は無人時間帯を挟む場合も、記録作成者と受領確認者を区別する。

## 5. 開始条件

- [ ] 引継ぎ対象、予定時刻、引継ぎ者及び受領者を特定した
- [ ] 受領者の資格、教育、配置、入館及び情報アクセス権を確認した
- [ ] 引継ぎ記録、関連台帳、案件一覧及び物件別手順を参照できる
- [ ] 対象状態、受渡物及び継続案件を照合できる時間と手段を確保した
- [ ] 緊急事象中の場合、現指揮者が移管方法と移管時点を決定した

## 6. 引継ぎ対象

| 区分 | 主な内容 | 必須確認 |
|---|---|---|
| 実施体制 | 出欠、配置、担当範囲、応援、連絡先役割 | 空白・重複・資格条件 |
| 対象状態 | 清掃状態、設備運転、警報、施錠、区域状態 | 現在状態と確認時刻 |
| 実施結果 | 完了、未実施、中断、要観察、異常 | 記録と現場状態の一致 |
| 継続案件 | 不具合、苦情、事故、工事、監視、再確認 | 案件ID、現在状態 |
| 次の行動 | 実施内容、期限、確認時刻、上申条件 | 担当者と代替者 |
| 物品・権限 | 鍵、端末、工具、測定器、薬剤、許可、アクセス権 | 番号、数量、状態、有効期限 |
| 制約・危険 | 立入制限、停止、火気、衛生、第三者影響 | 禁止事項と解除条件 |
| 臨時指示 | 作業変更、重点確認、顧客指示 | 指示元、承認、適用期限 |

## 7. 標準手順

| No. | 実施内容 | 完了基準 | 不一致時 |
|---:|---|---|---|
| 1 | 引継ぎ者・受領者・対象・時刻を記録する | 責任移管の対象範囲が明確 | 現場責任者へ確認 |
| 2 | 記録と対象の現在状態を照合する | 確認時刻を含め一致 | 相違を新規案件化 |
| 3 | 完了・未完了・中断・要観察・異常を区別する | 状態と根拠が明確 | 推測で完了扱いしない |
| 4 | 継続案件を一件ずつ説明し、受領者が復唱する | 担当、期限、次の行動、上申条件が明確 | 担当未定は受領保留 |
| 5 | 物品、鍵、装備、資料及び権限を照合する | 数量、状態、権限が一致 | 利用停止・紛失等の初動 |
| 6 | 制約、危険、区域状態及び臨時指示を確認する | 根拠、期限、解除条件が明確 | 責任者・専門担当へ確認 |
| 7 | 受領判定を行う | 開始可否と未解決事項が明確 | 8章に従い処理 |
| 8 | 双方が記録を確定し、責任移管時刻を記録する | 説明と受領が双方確認済み | 移管未成立として扱う |
| 9 | 受領後の初回確認を行う | 重要状態と記録が一致 | 相違を上申・案件化 |

## 8. 受領判定

| 状態 | 条件 | 標準対応 |
|---|---|---|
| 受領可 | 必要な人員、情報、物品、権限が揃い、継続案件の担当・期限が明確 | 責任移管を記録する |
| 条件付き受領 | 軽微な不足に代替策、期限、承認者がある | 条件を記録して移管する |
| 受領保留 | 重要情報不足、状態不一致、担当不明、物品・権限不備がある | 是正又は上位判断まで移管しない |
| 受領不可 | 資格・配置条件未達、重大安全問題、指揮不明、受領権限なし | 業務開始せず緊急上申する |

引継ぎ者の説明完了、勤務終了又は退出だけでは責任移管は成立しない。受領者による受領確認と移管時刻の記録を必要とする。

## 9. 異常時対応

| 事象 | 初動 | 後続 |
|---|---|---|
| 引継ぎ者不在 | 記録、現場、上位責任者から事実を再構成 | 欠落を明示し条件付き受領等を判断 |
| 受領者不在・不適格 | 代替者・応援を要請 | 配置変更又は業務縮小を上位判断 |
| 記録と現場の不一致 | 安全側へ状態を置き、対象を再確認 | 相違を案件化し訂正履歴を残す |
| 物品・鍵・権限不一致 | 使用を止め、影響範囲を確認 | 領域別の紛失・故障・権限手順へ |
| 緊急事象継続中 | 現指揮者の指示、人命保護、通信継続を優先 | 指揮、現場、記録を明示的に移管 |
| 個人・機密情報の誤伝達 | 伝達を停止し範囲を保全 | 責任者・情報管理手順へ接続 |

## 10. 権限・上申

| 判断・行為 | 権限 | 標準的な扱い |
|---|---|---|
| 記録・現物・状態の照合 | L0 | 引継ぎ者・受領者が実施 |
| 軽微な確認追加・代替物品使用 | L1 | 承認済み範囲で実施 |
| 条件付き受領、担当・配置変更 | L2 | 現場責任者が承認 |
| 追加要員、契約外対応、費用発生 | L3 | 契約・予算権限者が決定 |
| 恒常的な運用変更、重大リスク受容 | L4 | オーナー等が決定 |
| 緊急停止・退避・通報 | 緊急権限 | 切迫時は引継ぎより優先 |
| 指揮権・施設利用判断の移管 | 現指揮者・権限者 | 受領者と移管時刻を明示する |

## 11. 記録・完了条件

- [ ] 引継ぎ者、受領者、対象範囲及び開始・受領時刻を記録した
- [ ] 対象状態、実施結果及び未実施・中断理由を記録した
- [ ] 継続案件ごとに案件ID、担当、期限、次の行動及び上申条件を記録した
- [ ] 物品・鍵・装備・資料・権限の照合結果を記録した
- [ ] 不足・相違、代替策、期限及び承認者を記録した
- [ ] 双方が受領結果を確認し、責任移管時刻を確定した
- [ ] 移管後の初回確認又は確認予定を記録した

未解決事項が残っていても、その所有者、期限、次の行動及び上申条件が明確で、権限者が条件付き受領を承認した場合は引継ぎを完了できる。未解決事項そのものを完了扱いにはしない。

## 12. 領域別手順との境界

| 文書 | 正とする内容 |
|---|---|
| 本手順 | 受領判定、継続案件の所有者、責任移管の成立条件 |
| 清掃手順 | 区画別の清掃状態、資機材・薬剤、未実施箇所、衛生上の注意 |
| 設備手順 | 運転状態、警報、停止・隔離、操作権限、計測値、復旧条件 |
| 警備手順 | 配置、鍵・装備、入退館、監視、事件・事故、警備指揮 |

領域別手順は本手順を省略するものではなく、領域固有の確認項目を追加する。重複記録を避ける場合も、本手順の完了条件を領域別帳票で追跡できるようにする。
