# PROC-COM-003 作業区域設定・解除

| 項目 | 内容 |

| 領域 | 共通 |
| 版 | 0.1 |
| 状態 | 初版草案 |
| 作成日 | 2026-07-22 |

> 本文書は清掃・設備・警備に共通する標準手順である。個別物件の作業許可、避難計画、消防計画、警備計画、動線計画及び法令上の立入規制を優先する。区域解除は作業終了時刻の到来だけで自動的に行わない。

## 1. 目的

作業、点検、異常対応等に伴う危険源から利用者、第三者及び作業者を保護するため、必要な範囲、時間及び方法で作業区域を設定・維持し、危険除去と原状回復を確認した後に権限者が解除する。作業完了、技術的復旧、区域解除及び施設利用再開を別状態として扱い、過剰な閉鎖と早すぎる解除の双方を防ぐ。

| 関連業務 | `BM-17-05`〜`BM-17-06` | 安全手順、危険予知・リスク評価 | 危険源と必要な防護レベルを決定する |

## 3. 適用範囲

### 3.1 適用対象

- 濡れ床、薬剤、清掃機械、工具、資材等が第三者へ危険を及ぼす清掃
- 開放盤、回転体、漏水、落下物、熱源、騒音等を伴う設備作業
- 事故・事件・災害現場の保全、立入制限及び避難動線確保
- 高所、火気、停電、断水、搬出入等の作業許可に区域管理が指定される作業
- 作業中断・無人時間を含め、危険源が残る区域

### 3.2 適用除外・禁止事項

- 緊急退避や公的機関による規制を遅らせること
- 承認された避難経路、消防設備、救護動線又はバリアフリー動線を無断で塞ぐこと
- 表示やカラーコーンだけで、危険の強さに対して不十分な防護とすること
- 作業者不在で、区域の監視・巡視・連絡体制を設けないこと
- 危険源、工具、薬剤、漏れ、開口部等が残る状態で解除すること
- 作業者の完了申告だけで、解除権限者の確認なく利用再開させること

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 作業責任者 | 危険源、区域範囲、時間、設備、代替動線及び監視方法を計画する |
| 区域設置者 | 承認された計画に従い、表示・柵・施錠等を設置して記録する |
| 作業者 | 区域を維持し、状態変化・第三者侵入・設備損傷を報告する |
| 区域監視者 | 人の誘導、巡視、無人時間の確認、緊急連絡を行う |
| 施設・警備担当 | 利用者動線、避難、営業、入退館及び対外周知を調整する |
| 解除確認者 | 危険除去、原状回復及び関係者確認後に区域解除を承認する |
| 利用再開権限者 | 施設運用上の利用再開を判断する。解除確認者と異なる場合がある |

## 5. 開始条件

- [ ] 作業内容、危険源、影響範囲、予定時間及び中断時の状態を確認した
- [ ] 作業許可、入館、停止、火気等の必要承認を得た
- [ ] 利用者、テナント、関連作業及び避難・救護動線への影響を確認した
- [ ] 区域範囲、出入口、表示、防護設備、代替動線及び監視方法を決定した
- [ ] 設置・維持・解除の責任者と緊急連絡経路を決定した
- [ ] `PROC-COM-002`に従い、作業前安全確認を完了した

## 6. 防護方法の選定

| 危険・条件 | 防護例 | 追加確認 |
|---|---|---|
| 軽微な一時危険 | 注意表示、カラーコーン、誘導 | 視認性、転倒防止、短時間監視 |
| 接触・転倒・飛散のおそれ | バー、柵、養生、迂回 | 子ども、視覚障害者、混雑時の有効性 |
| 重大危険・立入禁止 | 固定柵、施錠、監視者、入室権限制御 | 侵入防止性能、非常時開放 |
| 設備・エネルギー危険 | 停止・隔離、ロックアウト等 | 資格、系統、誤操作防止、復旧権限 |
| 事故・事件現場 | 広域規制、現場保存、公的機関との境界 | 証拠保全、救護・消防活動の優先 |
| 夜間・無人時間 | 照明、遠隔監視、定期巡視、連絡表示 | 停電・通信断、翌朝利用前確認 |

防護方法は、危険の強さ、利用者特性、混雑、視認性、時間及び無人化を考慮して選定する。

## 7. 標準手順

| No. | 実施内容 | 完了基準 |
|---:|---|---|
| 1 | 危険源、第三者動線、設備系統及び周辺作業を現地確認する | 計画との差が明確 |
| 2 | 区域境界、出入口、退避・代替動線を確定する | 必要最小限かつ危険を包含 |
| 3 | 表示、柵、施錠、養生、照明等を設置する | 全接近方向から認識・防護できる |
| 4 | 避難、消防、救護、搬送及び施設運用への支障を確認する | 代替動線と緊急時対応が有効 |
| 5 | 設置状態、時刻、範囲、責任者を記録し作業開始を許可する | 設定完了が関係者確認済み |
| 6 | 作業中、交代時、中断時及び状態変化時に区域を巡視する | 防護設備と表示が有効 |
| 7 | 範囲・危険・時間の変更時に再評価し、区域を変更・再周知する | 変更履歴と承認が明確 |
| 8 | 作業終了後、危険除去、工具・薬剤・廃材、設備状態を確認する | 残留危険と未完了が明確 |
| 9 | 必要な技術・品質・警備・施設確認を得る | 解除条件を満たす |
| 10 | 解除承認後に防護設備を安全な順序で撤去し、原状回復する | 区域解除を記録 |
| 11 | 必要に応じ利用再開権限者へ引き渡す | 利用状態と制限が明確 |

## 8. 維持・変更・中断

| 事象 | 標準対応 |
|---|---|
| 表示・柵の移動、破損 | 作業を安全側へ止め、直ちに復旧。原因と影響を記録 |
| 第三者の侵入・接近 | 安全に誘導し、危険度に応じ作業中止・区域拡大・警備要請 |
| 作業範囲・危険源の拡大 | 再リスク評価し、範囲・防護・周知・承認を更新 |
| 作業中断・無人化 | 危険源を安全化するか、区域を継続して監視・巡視条件を設定 |
| 避難・緊急活動の発生 | 人命保護と公的機関・防災指揮を優先し、区域を再編 |
| 予定時刻超過 | 自動解除せず、延長承認、周知及び交代者への引継ぎを行う |

## 9. 解除判定

| 状態 | 条件 | 扱い |
|---|---|---|
| 解除可 | 危険源除去、原状回復、残置物なし、必要確認・承認済み | 区域を解除し記録 |
| 条件付き解除 | 一部制限を残して安全に利用可能で、範囲・期限・監視・承認が明確 | 制限を表示し部分解除 |
| 解除保留 | 確認未了、設備状態不明、乾燥・冷却待ち、関係者未確認 | 区域を維持 |
| 解除不可 | 危険・漏れ・開口・未隔離・事件現場保全等が継続 | 立入禁止を維持し上申 |

作業完了や警報消去だけでは解除可としない。技術的復旧と施設利用再開が別判断の場合は、それぞれの状態・判断者を記録する。

## 10. 権限・上申

| 判断・行為 | 権限 | 標準的な扱い |
|---|---|---|
| 承認済み計画どおりの設置・巡視・撤去 | L0 | 教育を受けた作業者 |
| 切迫危険時の一時閉鎖・作業中止 | 緊急権限 | 直ちに実施し責任者へ報告 |
| 軽微な範囲拡大、誘導変更 | L1 | 承認済み範囲内 |
| 部分閉鎖、臨時動線、条件付き解除 | L2 | 現場・施設・警備責任者 |
| 営業・長時間利用制限、追加費用 | L3 | 契約・施設・予算権限者 |
| 長期閉鎖、重大リスク受容 | L4 | オーナー等 |
| 技術的安全確認 | 専門権限 | 資格・操作権限を持つ者 |
| 施設利用再開 | 施設運用権限 | 区域解除と分けて判断 |

## 11. 記録・完了条件

- [ ] 区域の目的、危険源、範囲、出入口、設定・解除時刻を記録した
- [ ] 表示、防護設備、代替動線及び設置状態の証跡を残した
- [ ] 作業・区域責任者、監視・巡視条件及び交代を記録した
- [ ] 変更、中断、第三者侵入、破損及び是正を記録した
- [ ] 危険除去、原状回復、残置物及び設備状態を確認した
- [ ] 解除判定、確認者、承認者、残る制限及び期限を記録した
- [ ] 施設利用再開の状態と判断者を区域解除と分けて記録した

本手順は、区域が解除された場合、又は区域継続時に次の責任者、巡視条件、期限及び解除条件が引き継がれた場合に完了できる。
