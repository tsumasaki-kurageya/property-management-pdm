# 衛生現場作業手順

## 1. 目的

本ディレクトリは、空気環境、水質、給水・空調衛生設備、貯水槽、排水設備・排水槽及びねずみ・昆虫等の防除について、測定・点検・作業・判定・報告を現場で追跡できる標準手順へ分解する。

上位ルールは[ビルメンテナンス現場作業手順](../README.md)に従う。法定基準の数値、周期、資格、行政報告及び保存期間は本文へ固定せず、実施時点の法令、自治体・保健所の取扱い、物件別衛生管理計画及び法定業務マスターを参照する。

## 2. 対象範囲

- 空気環境の測定、基準・傾向判定、異常時の設備・運用への引渡し
- 給水・空調衛生設備の衛生点検
- 水質の採水・測定・検査、異常時の給水影響評価
- 貯水槽の清掃・消毒、設備復旧、水質・給水再開確認
- 排水設備・排水槽の点検・清掃・汚泥等の引渡し・復旧
- ねずみ・昆虫等の調査、防除、効果判定、再発防止
- 衛生記録、資格者確認、顧客・行政報告及び保存への引渡し

## 3. 手順の分割基準

| 分解軸 | 分割する差 |
|---|---|
| 目的 | 測定、点検、清掃・消毒、防除、報告 |
| 危険源 | 酸欠・閉所、感染、薬剤、給水汚染、第三者曝露 |
| 成果物 | 測定結果、点検結果、清掃・復旧記録、防除効果、法定帳簿 |

## 5. 共通設計原則

- 測定・点検結果と作業条件を一体で記録し、基準外値を削除・平均化して正常化しない。
- 測定・清掃・防除の実施、設備復旧、衛生適合、給水・施設利用再開を別状態・別権限として扱う。
- 酸欠・閉所、感染、薬剤、給水汚染等の専門危険は一般手順で権限を拡張せず、承認済み作業計画・資格者・登録事業者へ接続する。
- 異常・要観察は、担当、期限、再確認条件及び終了条件が受領されるまで閉じない。
- 手順本文と、物件別測定点・基準値・薬剤・連絡網・実行記録用チェックリストを分離する。

## 6. 判定と権限

| 状態 | 衛生領域での意味 | 標準対応 |
|---|---|---|
| 正常 | 適用基準・作業条件・証跡が整合 | 記録して完了 |
| 要観察 | 直ちに基準外ではないが傾向・再発・疑義がある | 期限付き再測定・再点検 |
| 異常 | 基準外、汚染、機能不良、効果不足等 | 安全確保、上申、是正、再確認 |
| 危険 | 健康影響、酸欠、有害ガス、重大汚染・曝露のおそれ | 中止・退避・区域設定・緊急連絡 |

緊急の停止・立入防止は現場で実施できる場合でも、設備復旧、衛生適合、給水・施設利用再開、行政報告及び重大リスク受容は所定の権限者が判断する。
