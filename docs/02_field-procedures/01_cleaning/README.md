# 清掃現場作業手順

## 1. 目的

本ディレクトリは、日常清掃、定期・特別清掃、廃棄物処理、消耗品補充及び清掃品質確認について、現場で再現可能な標準手順へ分解するための方針を定める。

上位ルールは[ビルメンテナンス現場作業手順](../README.md)に従う。本READMEは清掃領域の対象、手順の分割単位、初期手順一覧、固有の判断・記録事項及び作成順を定義する。

## 2. 対象範囲

### 2.1 対象

- 共用部、トイレ、給湯室、エントランス、外周等の日常清掃
- 廃棄物の回収、分別、集積及び所定場所への引渡し
- 衛生消耗品等の残量確認、補充及び不足報告
- 床面等の定期・特別清掃
- 作業者による仕上がり確認、責任者検査及び再清掃
- 清掃中に発見した破損、漏水、忘れ物、汚染等の一次対応と上申

### 2.2 対象外

- 害虫防除、空気環境測定、水質検査等の専門的な衛生管理手順
- 許可・委託契約・マニフェスト等を含む産業廃棄物処理の法定実務全体
- 高所ガラス、石綿、感染性物質等、固有の資格・作業計画を要する専門作業の詳細
- 個別薬剤・機材の取扱説明書を置き換える操作説明

対象外業務と接続する場合は、一般清掃手順へ詳細を取り込まず、適用条件、作業中止条件及び専門手順への引渡しを明示する。

## 3. 手順の分割基準

清掃手順は、次のいずれかが大きく異なる場合に分割する。

| 分解軸 | 分割を検討する差 |
|---|---|
| 対象場所 | 利用者動線、衛生区分、立入制限、什器構成 |
| 汚れ・廃棄物 | 通常汚れ、油脂、体液等の汚染、鋭利物、危険物 |
| 材質 | 石材、弾性床、木質、カーペット、金属、ガラス等 |
| 作業方式 | 拭き、掃き、吸引、洗浄、剥離、塗布、回収等 |
| 資機材・薬剤 | 誤使用時の損傷・健康影響、必要な保護具、希釈・保管方法 |
| 区域管理 | 営業中作業、全面閉鎖、部分規制、第三者立入への対応 |
| 品質基準 | 外観、衛生、残留水分、滑り、臭気、補充状態等 |
| 実施周期 | 日常、定期、臨時、緊急対応 |

建物用途、常駐・巡回、元請け・再委託等の違いだけで手順全体を複製せず、原則として適用差分に記載する。

## 5. 共通手順との接続

各清掃手順は、必要に応じて共通手順を前後に接続する。

| 工程 | 共通手順 | 清掃領域での主な確認 |
|---|---|---|
| 作業前 | [`PROC-COM-002` 作業前安全確認・危険予知](../00_common/PROC-COM-002_pre-work-safety-check.md) | 薬剤、濡れ床、電源、第三者動線、汚染、重量物 |
| 区域設定 | [[`PROC-COM-003` 作業区域設定・解除](../00_common/PROC-COM-003_work-area-control.md)](../00_common/PROC-COM-003_work-area-control.md) | 表示、通行規制、代替動線、利用者への配慮 |
| 異常発見 | [[`PROC-COM-004` 異常上申](../00_common/PROC-COM-004_abnormality-escalation.md)](../00_common/PROC-COM-004_abnormality-escalation.md) | 破損、漏水、害虫、忘れ物、危険物、感染リスク |
| 完了 | [[`PROC-COM-005` 完了報告](../00_common/PROC-COM-005_completion-reporting.md)](../00_common/PROC-COM-005_completion-reporting.md) | 仕上がり、原状回復、未実施、再清掃、写真 |

共通手順を参照しても、個別手順側には当該作業に固有の開始条件、中止条件、品質基準及び完了条件を記載する。

## 6. 清掃固有の記述規則

### 6.1 作業順序・衛生管理

- 高所から低所、清潔側から汚染側等の作業順序を、対象場所と衛生区分に合わせて明示する。
- 用具の区域別・用途別の使い分けと、使用後の洗浄、乾燥、保管を定義する。
- 薬剤は対象材質、汚れ、濃度、接触時間、換気及び混合禁止事項を確認し、SDSとメーカー指定を優先する。
- 汚染の種類が不明、鋭利物がある、健康影響が疑われる等の場合は、通常清掃を中止して安全確保と上申を行う。

### 6.2 判定基準

| 状態 | 清掃領域での例 | 標準対応 |
|---|---|---|
| 正常 | 契約・物件別の仕上がり基準を満たし、利用再開に支障がない | 記録して完了 |
| 要観察 | 軽微な変色、繰返し発生する汚れ、消耗品の減りが早い | 記録し、次回確認又は傾向確認へつなぐ |
| 異常 | 清掃漏れ、除去できない汚れ、破損、漏水、在庫不足 | 再清掃、一次対応、責任者報告又は別業務化 |
| 危険 | 滑り・転倒、薬剤曝露、鋭利物、危険物、感染のおそれ | 作業中止、立入防止、安全確保、緊急連絡 |

「きれいになった」だけで完了とせず、対象ごとに外観、臭気、残留水分、付着物、補充、什器位置及び利用再開条件を定義する。定性的基準は、物件別の写真例又は限度見本で補う。

### 6.3 権限境界

- L0：承認済み手順による清掃、補充、記録
- L1：承認済み範囲での再清掃、軽微な作業順変更、安全確保
- L2以上：仕様外の薬剤・工法、予定外の区域閉鎖、追加作業、利用制限、廃棄判断等

緊急の立入防止は現場で実施できる場合でも、区域の利用再開、施設利用制限、契約外作業及び追加費用は所定の権限者へ上申する。

## 7. 記録・証跡

上位READMEの共通記録に加え、次を対象に応じて記録する。

- 清掃対象、材質、汚れ又は廃棄物の区分
- 使用した機材・薬剤、必要な場合は濃度・使用量
- 区域設定・解除時刻、利用制限及び代替動線
- 清掃前後、除去不能箇所、破損・漏水等の写真
- 仕上がり判定、検査者、再清掃の要否と結果
- 回収量・分別不良・危険物等の発見及び引渡し
- 消耗品の補充数、残数、不足及び発注・引継ぎ先

## 8. 適用差分

| 分析軸 | 清掃手順へ反映する主な差 |
|---|---|
| 建物用途 | 衛生水準、重要区画、営業時間、利用者属性、臭気・騒音制約 |
| 管理方式 | 資機材の現地保管又は持参、巡回時の滞在時間、完了写真、再訪判断 |
| 契約役割 | 作業指示、自主検査、元請検査、不適合の差戻し、廃棄物の引渡し |
| 法令義務 | 廃棄物、薬剤、作業安全、記録保存等の適用条件と義務主体 |
| 責任分界 | 区域閉鎖、利用制限、追加清掃、設備不具合化、追加費用の決定先 |

差分は上位READMEで定めた既存プロファイルを参照し、個別物件の仕様、清掃基準、ゾーニング及び連絡網へ具体化する。

## 9. 作成順と完了条件

初期作成は次の順序とする。

1. `PROC-CLN-002` トイレ清掃
2. `PROC-CLN-001` 共用部日常清掃
3. `PROC-CLN-006` 清掃品質検査・再清掃
4. `PROC-CLN-003` 廃棄物回収・分別
5. `PROC-CLN-004` 消耗品確認・補充
6. `PROC-CLN-005` 定期床面清掃

本領域の初期整理は、各手順について主業務ID、開始・完了条件、標準工程、品質判定、異常分岐、権限、記録及び適用差分が定義され、共通手順と重複・欠落なく接続できた状態を完了とする。
