# PROC-CLN-001 共用部日常清掃

> 本文書は標準手順である。個別物件の契約仕様、対象材質、清掃基準、薬剤・機材指定、作業時間、区域規制及び廃棄物ルールを優先する。

## 1. 目的

玄関、ロビー、廊下、階段、エレベーターホール等の共用部を、利用者の安全と施設利用を妨げない方法で清掃し、契約上の美観・衛生・安全水準を維持する。清掃中に発見した破損、漏水、危険物、遺失物等を適切な後続業務へ引き渡す。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-06-03` | 日常清掃を実施する | 共用部の日常清掃を実施する |
| 関連業務 | `BM-13-01`〜`BM-13-05` | 作業記録・完了・確認 | 実施、異常、未実施を記録・報告する |
| 関連業務 | `BM-17-01`〜`BM-17-04`、`BM-17-06` | 安全・品質・リスク管理 | 安全、品質及び危険予知を行う |

## 3. 適用範囲

### 3.1 適用対象

- 玄関、風除室、ロビー、廊下、階段、エレベーターホール
- 契約仕様で指定された共用の床、扉、手すり、什器、ガラスの手の届く範囲
- 日常的な除塵、拭き、拾い掃き、汚れ除去及び軽微な整理

### 3.2 適用除外・使用禁止条件

- 高所、外壁、機械内部、電気設備内部等の専門作業
- 血液、薬品、感染性物質、鋭利物、危険物等の未教育処理
- 材質又は薬剤適合が確認できない汚れ除去
- 大量漏水、破損、剥離、段差等により通常清掃で安全を確保できない状態
- 私有物・展示物・機密物の無断移動

## 4. 実施体制

| 役割 | 主な責任 | 必要な力量 | 単独実施可否 |
|---|---|---|---|
| 清掃作業者 | 準備、清掃、品質確認、記録、異常報告 | 物件教育、材質・薬剤・機材、安全教育 | 物件条件による |
| 清掃責任者 | 指示、区域・仕様確認、品質判定、上申 | 契約仕様と品質基準の理解 | 該当なし |
| 施設・警備担当 | 動線、鍵、拾得物、危険区域等の連携 | 物件別権限 | 該当なし |
| 設備担当 | 漏水、設備破損、電気等の専門判断 | 対象に必要な力量 | 該当なし |

## 5. 開始条件

- [ ] 最新の作業指示、対象区画、時間帯、頻度及び品質基準を確認した
- [ ] 材質、薬剤・機材の適合、SDS、希釈・混合禁止事項を確認した
- [ ] 利用者動線、工事、催事、搬入、天候、床面状態を確認した
- [ ] 必要な区域表示、代替動線、鍵及び作業許可を確認した
- [ ] 用具の清潔区分、員数及び損傷を確認した
- [ ] `PROC-COM-002`による作業前安全確認を完了した

## 6. 必要資料・資機材

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 資料 | 契約仕様、作業表、材質表、品質基準、SDS | 最新版、対象範囲 |
| 清掃用具 | モップ、クロス、ほうき、ちり取り、掃除機等 | 清潔区分、損傷、対象材質 |
| 薬剤 | 物件指定の洗剤・除菌剤等 | ラベル、期限、希釈、混合禁止 |
| 安全用品 | 作業中表示、区画材、手袋、滑りにくい履物等 | 視認性、必要数、PPE適合 |
| 記録 | チェックリスト、端末、必要時カメラ | 時刻、保存先、撮影制限 |

## 7. 作業前確認・危険予知

| 危険・影響要因 | 予防措置 | 発生時の初動 |
|---|---|---|
| 濡れ床・転倒 | 小区画施工、表示、代替動線、乾燥確認 | 作業中止、立入制限、救護・報告 |
| 利用者との接触 | 混雑を避け、用具を動線外へ置く | 安全確保、状況確認、責任者連絡 |
| 薬剤誤使用 | ラベル、材質、希釈、換気、混合禁止を照合 | 使用中止、隔離、SDSに従う |
| 電源・機材 | コード、プラグ、漏電、通行交差を確認 | 電源遮断、使用中止、設備担当へ |
| 遺失物・危険物 | 素手で不用意に触れず物件ルールを確認 | 現場保全、警備・責任者へ引渡し |
| 破損・漏水 | 清掃前に状態を観察する | 拡大防止、区域管理、設備担当へ |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 指示と現場を照合し、作業順を決める | 対象・時間・動線が整合する | 不明区画は開始保留 | 開始、対象、指示ID |
| 2 | 作業表示を設置し、必要範囲を小さく区画する | 表示が全進入方向から見える | 代替動線を確保できなければ保留 | 設置時刻、必要時写真 |
| 3 | 私有物、遺失物、破損、漏水等を事前確認する | 清掃で状態を変える前に把握した | 触れず所定先へ連絡 | 場所、状態、連絡 |
| 4 | 上部・奥から手前、汚れの少ない箇所から順に除塵する | 汚れを再付着・拡散させない | 用具汚染時は交換 | 実施区画 |
| 5 | 材質に適合する方法で床・接触部を清掃する | 汚れ除去と材質保全を両立する | 変色・損傷時は直ちに中止 | 方法、薬剤、異常 |
| 6 | 汚れた用具・廃棄物を区分し回収する | 清潔用具と交差しない | 危険物は通常回収しない | 回収・引渡し |
| 7 | 仕上がり、残留薬剤、濡れ、臭気、置き忘れを確認する | 品質基準内で安全に利用できる | 再清掃又は利用保留 | 判定、未達 |
| 8 | 什器を許可された位置へ戻し、区域を解除する | 原状回復・乾燥・通行安全を確認 | 解除せず責任者へ | 解除時刻 |
| 9 | 結果、未実施、異常、用具の状態を報告する | 後続対応と期限が明確 | 受領まで追跡 | 完了、引継ぎ |

## 9. 判定基準

| 状態 | 手順固有の基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 正常 | 指定範囲を実施し、目立つ汚れ、塵埃、濡れ、残留物及び原状不良がない | 記録して完了 | 可 |
| 要観察 | 軽微な落ちにくい汚れ、反復汚染、材質劣化等がある | 記録し再確認・別作業を提案 | 条件付き可 |
| 異常 | 仕様未達、薬剤・機材不適合、破損、漏水、害虫、遺失物等がある | 所定担当へ引渡し | 原則不可 |
| 危険 | 転倒、感電、危険物、感染、重大破損等のおそれがある | 作業中止、区域管理、緊急連絡 | 不可 |

定性的品質は契約仕様、写真例又は限度見本で補い、個人の印象だけで合否を決めない。

## 10. 異常時対応

| 事象 | 安全確保・一次対応 | 報告・後続 |
|---|---|---|
| 落ちない汚れ・材質劣化 | 無理な擦り・強薬剤を使用しない | 定期・特別清掃等を検討 |
| 破損・漏水 | 周囲を規制し状態を変えない | 施設・設備担当へ引渡す |
| 遺失物 | 物件ルールに従い必要最小限で取り扱う | 警備等へ時刻・場所とともに引渡す |
| 鋭利物・体液・危険物 | 素手で触れず区域を確保する | 専門手順・責任者へ接続する |
| 薬剤飛散・体調不良 | 使用中止、退避、換気等をSDSに従い実施 | 救護、責任者、必要時緊急機関へ |
| 利用者事故 | 二次災害を防ぎ訓練範囲で救護する | 施設・警備責任者へ、事実を記録 |

## 11. 権限・上申

| 判断・行為 | 権限 | 標準的な扱い |
|---|---|---|
| 承認済み方法での日常清掃 | L0 | 清掃作業者が実施 |
| 小区画化、再清掃、安全な用具交換 | L1 | 承認済み範囲で実施・記録 |
| 作業時間・工法変更、区域閉鎖、仕様外薬剤 | L2 | 清掃・施設責任者の承認 |
| 契約外作業・追加費用 | L3 | 契約・予算権限者が決定 |
| 長期利用制限・改修 | L4 | オーナー等が決定 |
| 危険時の中止・立入防止 | 緊急権限 | 直ちに実施し報告 |
| 区域利用再開 | L1〜L2 | 乾燥・安全確認後、物件権限に従う |

## 12. 作業後確認・完了条件

- [ ] 指定区画と実施結果、未実施理由を記録した
- [ ] 仕上がり、乾燥、残留薬剤、臭気、置き忘れを確認した
- [ ] 什器、扉、照明、鍵等を所定状態へ戻した
- [ ] 用具を洗浄・区分・保管し、薬剤・廃棄物を所定方法で処理した
- [ ] 異常、遺失物、破損、要観察事項を所定担当へ引き渡した
- [ ] 作業表示を安全確認後に撤去した
