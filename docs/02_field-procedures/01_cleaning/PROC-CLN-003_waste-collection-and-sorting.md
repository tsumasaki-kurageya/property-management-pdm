# PROC-CLN-003 廃棄物回収・分別

## 0. 文書管理

| 項目 | 内容 |
|---|---|
| 手順ID | `PROC-CLN-003` |
| 手順名 | 廃棄物回収・分別 |
| 領域 | 清掃 |
| 版 | 0.1 |

| 作成日 | 2026-07-22 |

> 本文書は標準手順である。個別物件の廃棄物区分、回収経路、集積場所、搬出時刻、危険物対応、委託契約及び自治体ルールを優先する。

## 1. 目的

所定の廃棄物を安全に回収し、物件別の分別基準に従って集積・引渡しを行う。分別不良、漏出、鋭利物、危険物、機密物等を通常廃棄物と混同せず、回収完了と処理責任の引渡しを追跡可能にする。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-06-06` | 廃棄物を回収・分別する | 回収、分別、集積、引渡しを行う |
| 関連業務 | `BM-13-02`〜`BM-13-04` | 結果・写真・完了報告 | 回収結果と例外を記録する |
| 関連業務 | `BM-13-11` | 異常を速報・上申する | 危険物等を責任者へ引き渡す |
| 関連業務 | `BM-17-05`〜`BM-17-07` | 安全・リスク・事故管理 | ばく露、漏出、負傷を管理する |
| 関連業務 | `BM-17-08`〜`BM-17-09` | 法令・証跡管理 | 法定対象の処理実務へ接続する |

## 3. 適用範囲

- 共用部、テナント指定場所等からの廃棄物回収
- 承認済み区分による分別確認、袋・容器の封緘
- 所定集積場所への搬送、数量・状態の記録、受領者への引渡し
- 分別不良、漏出、異臭、鋭利物、危険物、機密物等の例外対応

産業廃棄物の分類判断、委託契約、収集運搬・処分、マニフェスト等の法定実務全体は対象外とし、適格な責任者・事業者へ引き渡す。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 回収作業者 | 対象照合、安全確認、回収、分別、搬送、記録 |
| 清掃責任者 | 例外判定、回収中止、再分別、上申、引渡し統制 |
| 排出部門・テナント | 排出区分、機密・危険物情報、分別不良の是正 |
| 廃棄物管理責任者 | 法令・契約適合、保管、許可業者・処理工程への引渡し |
| 収集運搬等の受領者 | 契約範囲での受領と後続処理 |

## 5. 開始条件

- [ ] 対象区域、回収時刻、回収対象、除外物及び経路を確認した
- [ ] 分別基準、容器・表示、集積場所及び引渡し先を確認した
- [ ] 必要な保護具、運搬具、漏出対策用品を準備した
- [ ] エレベーター、利用者動線、施錠等の使用条件を確認した
- [ ] 危険物・鋭利物・液漏れ等を発見した場合の中止・連絡条件を確認した

## 6. 必要資料・用具

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 基準 | 分別表、対象外一覧、自治体・物件ルール | 最新版、対象場所 |
| 容器 | 指定袋、容器、ラベル、封緘具 | 区分、破損、容量 |
| 運搬 | カート、台車、養生 | 積載量、転倒・落下防止 |
| 安全 | 手袋、保護具、吸収材、表示 | 廃棄物の性状に適合 |
| 記録 | 回収表、秤、撮影機器 | 単位、個人・機密情報 |

## 7. 作業前確認・危険予知

| 危険・影響 | 予防措置 | 発生時 |
|---|---|---|
| 鋭利物・破袋 | 外観確認し、押し込み・手探りをしない | 接触を止め区域管理・上申 |
| 液漏れ・粉じん・異臭 | 容器状態と換気を確認 | 接近・運搬を中止し安全確保 |
| 重量物・転倒 | 重量、積載、経路、段差を確認 | 無理に持たず応援・機材を要請 |
| 感染性・薬品・電池等 | 通常区分と分離し表示を確認 | 専門手順・責任者へ引渡し |
| 機密・個人情報 | 中身を閲覧せず指定容器で扱う | 露出・混入を記録し責任者へ |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 指示、区域、回収点、経路、集積先を照合する | 対象と責任範囲が明確 | 不明点は開始保留 | 指示ID、時刻 |
| 2 | 容器・袋の外観、表示、漏れ、突出物を確認する | 安全に扱える | 危険なら触れず区域管理 | 場所、状態、写真 |
| 3 | 指定方法で回収し、置換容器・袋を設置する | 周辺を汚染せず回収済み | 漏れ・散乱は拡大防止 | 数量、交換状態 |
| 4 | 分別表示と内容の外観を照合する | 基準と一致 | 手探りせず隔離・再分別判断 | 区分、不一致 |
| 5 | 容量・重量を超えず封緘・表示する | 漏れ・落下がない | 分割又は適合容器へ | 袋数、重量等 |
| 6 | 指定経路で集積場所へ搬送する | 動線・床面・設備を損なわない | 区域変更、応援要請 | 経路、搬送時刻 |
| 7 | 区分別の所定位置へ集積し状態を確認する | 混載、転倒、漏出、過量がない | 隔離し責任者へ | 集積位置、状態 |
| 8 | 数量・例外・未回収を記録し受領者へ引き渡す | 受領者、時刻、残課題が明確 | 受領不能は保管責任を設定 | 受領、案件ID |
| 9 | 回収点・経路・用具を清掃し区域を戻す | 二次汚染・残置がない | 区域維持・追加清掃 | 完了時刻、写真 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 正常 | 対象を区分どおり回収・集積し、漏出・残置がない | 記録し完了 | 可 |
| 要是正 | 安全に再分別・再封緘できる軽微な不一致 | 承認範囲で是正・再確認 | 是正後可 |
| 異常 | 危険物、機密物、反復する分別不良、集積超過等 | 隔離、上申、別案件化 | 引渡し成立時に条件付き可 |
| 危険 | 漏出、強い異臭、鋭利物、未知物、健康影響のおそれ | 接触・搬送中止、区域管理、緊急連絡 | 不可 |

## 10. 異常時対応

- 未知物、薬品、感染性物、膨張電池等は推測で分類・開封せず、`PROC-COM-004`へ接続する。
- 漏出・飛散・鋭利物等では`PROC-COM-003`により区域を管理する。
- 分別不良を是正する場合も、手探り、圧縮、無理な詰替えを行わない。
- 法定区分・排出者責任・処理方法の判断は、廃棄物管理責任者へ引き渡す。
- 負傷・ばく露時は作業を中止し、物件別緊急手順に従う。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 指定区分の回収・分別・記録 | L0 |
| 承認済み範囲の再分別・容器交換 | L1 |
| 危険物判定、予定外保管、回収拒否、区域閉鎖 | L2以上 |
| 契約変更、追加回収、処理費用判断 | L3 |
| 処理方針・重大リスク受容 | L4又は法的責任主体 |

## 12. 記録・完了条件

- [ ] 回収場所、区分、数量・重量等、時刻及び作業者を記録した
- [ ] 未回収、分別不良、漏出、危険物等の事実と安全措置を記録した
- [ ] 集積場所、保管状態、受領者及び受領時刻を記録した
- [ ] 回収点、経路、用具及び区域を原状回復した
- [ ] 例外案件の担当、期限、保管責任及び受領を確認した
- [ ] `PROC-COM-005`による完了報告へ接続した

回収場所から物がなくなっただけでは完了しない。適切な区分で安全に集積され、後続責任の引渡しが成立していることを完了条件とする。
