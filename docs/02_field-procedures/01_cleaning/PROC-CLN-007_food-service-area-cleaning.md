# PROC-CLN-007 給湯室・食堂清掃

| 項目 | 内容 |
|---|---|

| 手順名 | 給湯室・食堂清掃 |
| 領域 | 清掃 |
| 版 | 0.1 |
| 状態 | 初版草案 |

> 本文書は標準手順である。食品を直接扱う厨房の衛生管理、個別物件の衛生区分、使用薬剤、廃棄物区分、設備操作及び利用再開基準を置き換えない。

## 1. 目的

給湯室、休憩室及び食堂の共用部分から食品残さ、油脂、汚れ及び廃棄物を除去し、清潔区域と汚染区域を交差させずに利用可能な状態へ戻す。一般共用部清掃と異なり、食品由来汚染、害虫誘引、熱源・刃物及び洗剤残留を判断対象とする。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 本手順との関係 |
|---|---|---|
| 主業務 | `BM-06-03` | 食事関連区域の日常清掃を実施する |
| 関連業務 | `BM-06-06`〜`BM-06-08` | 廃棄物、補充及び結果記録へ接続する |
| 関連業務 | `BM-06-09`〜`BM-06-10` | 品質検査及び再清掃へ接続する |
| 関連業務 | `BM-13-02`〜`BM-13-04`、`BM-13-11` | 記録、完了報告及び異常上申を行う |
| 関連業務 | `BM-17-01`〜`BM-17-07` | 品質、安全及び不適合を管理する |

## 3. 適用範囲・除外

- 共用の給湯室、休憩室、食堂、流し台、テーブル及び床面
- 食品残さ、軽微な油脂、通常の飲料こぼれ及び一般廃棄物
- 承認済み方法による接触面の清拭、洗浄、乾燥及び補充

厨房設備内部、営業用調理設備、グリストラップ、未知の薬品、広範囲の腐敗物、感染性が疑われる汚染及び害虫防除は対象外とし、専門手順へ引き渡す。

## 4. 実施体制・開始条件

| 役割 | 主な責任 |
|---|---|
| 清掃作業者 | 対象確認、区域分け、清掃、乾燥、記録 |
| 清掃責任者 | 衛生区分、工法、中止、再清掃及び上申の判断 |
| 施設・衛生責任者 | 利用制限、専門対応及び利用再開の判断 |

- [ ] 利用時間、残置物、熱源・電源及び設備停止条件を確認した
- [ ] 食品接触面と床・廃棄物用の用具を区別した
- [ ] 薬剤の対象面、濃度、接触時間、すすぎ及び乾燥条件を確認した
- [ ] 廃棄判断を清掃作業者が独断で行わない範囲を確認した

## 5. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 |
|---:|---|---|---|
| 1 | 指示、対象区域、利用終了及び残置物を照合する | 清掃対象と所有物が区別できる | 不明な物は移動・廃棄せず確認する |
| 2 | 熱源、刃物、割れ物、漏電、腐敗及び害虫痕跡を確認する | 通常清掃を安全に開始できる | 接触を止め区域管理・上申する |
| 3 | 食品残さと廃棄物を指定区分で回収する | 飛散・漏出・混載がない | `PROC-CLN-003`へ接続する |
| 4 | 清潔側から汚染側へ、上部から下部へ清拭する | 用具の交差使用がない | 用具交換、再清掃を行う |
| 5 | 流し台、接触面及び床面を材質に適合した方法で洗浄する | 油脂、付着物、洗剤残留がない | 工法を変えず責任者へ上申する |
| 6 | 水分を除去し、排水状態、臭気、漏水を確認する | 乾燥し利用に支障がない | 漏水・排水不良は設備案件へ渡す |
| 7 | 什器、表示及び消耗品を復旧し品質確認する | 衛生区分と配置が復元されている | `PROC-CLN-006`へ接続する |
| 8 | 廃棄物、異常、未実施及び利用制限を記録する | 後続責任が明確 | 受領まで完了扱いにしない |

## 6. 判定・中止条件

| 状態 | 基準 | 対応 |
|---|---|---|
| 正常 | 食品残さ、油脂、付着物、残留水分及び異臭が基準内 | 記録して完了 |
| 要観察 | 反復汚れ、軽微な変色、排水の遅れ又は害虫誘引のおそれ | 期限付きで再確認 |
| 異常 | 腐敗、害虫痕跡、排水不良、設備破損又は除去不能汚れ | 上申し別案件化 |
| 危険 | 刃物、割れ物、漏電、未知物、強い腐敗又は健康影響のおそれ | 作業中止、区域管理、専門対応 |

## 7. 権限・完了条件

- L0：承認済み範囲の回収、清拭、洗浄、乾燥及び記録
- L1：仕様内の再清掃、用具交換、安全確保
- L2以上：設備停止、予定外閉鎖、薬剤・工法変更、専門対応及び利用再開
- L3以上：契約外作業、追加費用又は恒常的な仕様変更

清掃対象が基準を満たし、洗剤・水分・廃棄物が残らず、設備異常と未実施が引き渡され、指定権限者が利用再開可能な状態を受領した時点を完了とする。

- 食品残さ、油脂、廃棄物、害虫痕跡、漏水及び設備異常
- 作業前後、品質判定、利用制限、受領者及び再確認期限
