# PROC-CLN-013 汚染物発見時の初動対応

## 0. 文書管理

| 項目 | 内容 |
|---|---|
| 手順ID | `PROC-CLN-013` |
| 手順名 | 汚染物発見時の初動対応 |
| 領域 | 清掃 |
| 版 | 0.1 |
| 状態 | 初版草案 |
| 作成日 | 2026-07-24 |

> 本手順は、嘔吐物、体液、排泄物、未知物、鋭利物、有害物質等を発見した時の接触回避、区域管理、評価及び引渡しを扱う。汚染の同定、医療判断、除染、廃棄及び利用再開の専門判断を一般清掃員に代替させない。

## 1. 目的

汚染又は有害性が疑われる物を発見した際に、通常清掃を開始せず、人・区域・情報を保全する。承認済み範囲で処理可能か、専門対応へ移管するかを判定し、二次ばく露と拡散を防ぐ。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 本手順との関係 |
|---|---|---|
| 主業務 | `BM-06-05` | 通常外の特別清掃へ移る前の初動を行う |
| 関連業務 | `BM-06-03`、`BM-06-06`、`BM-06-08`〜`BM-06-10` | 日常清掃の中止、廃棄、記録、検査及び是正 |
| 関連業務 | `BM-11-06`、`BM-11-09` | 不審物又は事件・急病現場へ接続する |
| 関連業務 | `BM-13-11` | 異常を速報・上申する |
| 関連業務 | `BM-17-05`〜`BM-17-09`、`BM-17-11` | 安全、事故、法令、証跡及び区域管理 |

## 3. 適用範囲・除外

嘔吐物、血液等の体液、排泄物、未知の粉・液体、鋭利物、薬品容器、強い異臭及び感染・有害性が疑われる物の発見初動を対象とする。物質同定、警察・消防・医療の専門対応、資格を要する除染及び法定廃棄判断は対象外とする。

## 4. 開始条件

- [ ] 物へ接触せず、人を近づけない状態を確保した
- [ ] 発見場所、時刻、見た目、周辺者及び拡散範囲を把握した
- [ ] 施設、衛生、安全、警備等の連絡経路を確認した
- [ ] 一般作業者が処理可能な承認済み条件と専門移管条件を確認した
- [ ] 必要な保護具、処理キット、廃棄容器及び区域用品の有無を確認した

## 5. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 |
|---:|---|---|---|
| 1 | 接近・接触を止め、周辺者を離す | ばく露拡大を防げる | 退避範囲を広げ緊急連絡する |
| 2 | 区域を設定し、踏み込み・飛散・設備拡散を防ぐ | 境界と監視が維持される | `PROC-COM-003`を強化する |
| 3 | 触れずに場所、量、状態、周辺影響を記録する | 事実と推定を分けられる | 無理に近づかず記録可能範囲に留める |
| 4 | 人の負傷・急病、犯罪・不審物、薬品等の可能性を確認する | 優先すべき緊急手順が明確 | 警備・救急・消防等へ接続する |
| 5 | 責任者が一般処理可否と必要体制を判定する | 承認済み条件に完全一致 | 専門対応へ移管する |
| 6 | 一般処理可の場合のみ専用手順・用具で処理する | 保護具、工程、廃棄、接触時間が適合 | 中止し区域を維持する |
| 7 | 使用用具、廃棄物、周辺面及び作業者の状態を管理する | 二次汚染・ばく露がない | 安全・衛生責任者へ上申する |
| 8 | 検査と利用再開判断へ引き渡し、記録する | 判断者が証跡と残課題を受領 | 区域を解除しない |

## 6. 判定・中止条件

一般処理可は、物質・範囲・処理方法・保護具・廃棄方法・実施者権限がすべて承認済み条件に一致する場合に限る。一項目でも不明、広範囲、鋭利物、薬品、有害物、事件性、設備への流入又は健康影響がある場合は専門移管とする。

## 7. 権限・完了条件

発見・接触回避・区域設定・速報はL0〜L1で直ちに行う。一般処理可否、区域変更及び利用再開はL2以上、専門業者手配・追加費用はL3以上、重大リスク受容はL4とする。

初動の完了は汚染除去の完了と同じではない。安全な区域管理、事実記録、対応区分の決定及び後続責任者の受領を初動完了とし、除染・廃棄・品質確認・利用再開は別に完了を追跡する。

## 8. 記録・関連文書

発見場所・時刻・状態、接触者、区域、写真、連絡、判定根拠、保護具、処理又は移管、廃棄引渡し、検査及び利用再開を記録する。
