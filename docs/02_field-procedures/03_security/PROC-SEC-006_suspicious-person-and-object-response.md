> 本文書は標準手順である。個別物件の警備計画、危機管理計画、警察指示、爆発物・危険物対応、個人情報及び映像取扱規程を優先する。

## 1. 目的

不審者又は不審物を認知した際に、警備員と利用者の身体安全を最優先として距離、遮蔽物及び退路を確保する。観察事実に基づき応援・警察等へ連絡し、接近・追跡・移動等による危険拡大を避けながら、区域、証跡及び対応責任を引き渡す。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-06` | 事故・事件へ対応する | 不審者・不審物の初動を行う |
| 関連業務 | `BM-12-03`〜`BM-12-07` | 問合せ・苦情・緊急対応 | 利用者案内と施設対応へ接続する |

## 3. 適用範囲

- 無断立入、徘徊、威嚇、追随、盗難・破壊等が疑われる人物
- 所有者不明、放置、異臭、液漏れ、配線、粉末等がある物品
- 警報・映像・利用者通報から不審事象を認知した場合
- 安全確保、利用者誘導、応援・警察連絡、区域管理、証跡保全及び引渡し

単独での制止・追跡・身体拘束、所持品検査、不審物の接触・移動・開封・処理及び危険物の判定は対象外とする。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 第一認知者・警備員 | 距離・退路確保、観察、通報、利用者誘導、記録 |
| 監視担当 | 映像・警報確認、対象追跡情報、証跡保全 |
| 警備責任者 | 応援配置、区域、連絡、警備指揮、引渡し統制 |
| 施設責任者 | 施設利用制限、テナント・利用者対応、案件管理 |
| 警察・消防等 | 公的な危険排除、捜査、処理及び現場指揮 |

## 5. 開始条件

緊急時は書面確認を待たず、退避と110・119等への通報を優先する。

- [ ] 場所、対象、人・物の特徴、認知時刻及び情報源を確認した
- [ ] 暴力、凶器、爆発、火災、薬品、二次対象等の危険兆候を確認した
- [ ] 自身と利用者の退路、遮蔽物、安全距離及び応援を確認した
- [ ] 警察・消防、警備責任者、施設責任者への連絡方法を確認した
- [ ] 監視映像、入退館記録等の証跡を権限内で保全できる状態にした

## 6. 必要資料・装備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 連絡 | 緊急連絡網、施設住所、進入口 | 110・119等へ伝える内容 |
| 通信 | 無線、電話、非常通報装置 | 通信試験、秘匿性 |
| 区域 | コーン、バー、案内表示 | 危険区域へ接近せず設置可能か |
| 証跡 | 監視端末、時計、記録票 | 保存・閲覧・開示権限 |

## 7. 初動時の危険予知

| 危険 | 原則 |
|---|---|
| 暴力、凶器、複数人、追跡 | 単独接近・制止・追跡を避け、退避・警察連絡 |
| 不審物、液体、粉末、配線、異臭 | 触れず、動かさず、通信機器使用制限等は公的機関指示に従う |
| 群集、撮影、流言 | 安全な代替動線へ誘導し、未確認情報を断定しない |
| 対象見失い | 最終確認位置・方向・特徴を共有し、危険な捜索をしない |
| 証跡消失 | 身体安全を優先しつつ、権限内で映像・ログを保全 |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 安全な位置から対象、場所、行動・状態を観察する | 事実と推定を区別 | 危険兆候時は即退避・通報 | 時刻、特徴、情報源 |
| 2 | 距離、遮蔽物、退路を確保し利用者を遠ざける | 自身と第三者の安全が優先 | 接近不能は監視・区域拡大 | 動線、誘導先 |
| 3 | 応援、警備責任者、施設責任者へ速報する | 場所、危険、人数、必要支援を伝達 | 切迫時は承認前に110・119 | 発信・受領時刻 |
| 4 | 緊急基準に従い警察・消防へ通報する | 住所、進入口、対象、危険を復唱 | 通信不能は代替手段 | 相手、回答、指示 |
| 5 | 安全な場合のみ、物件計画の範囲で声掛け・立入保留を行う | 退路・応援・中止条件が成立 | 威嚇・逃走・凶器時は中止 | 発言、反応、時刻 |
| 6 | 不審物には触れず、周辺を区域管理する | 距離・範囲は公的機関指示を優先 | 異臭・液漏れ等は退避拡大 | 位置、外観、区域 |
| 7 | 映像、入退館、警報等を権限内で保全する | 原本性、時刻、取扱者が追跡可能 | 無断複製・外部開示しない | 証跡ID、保全者 |
| 8 | 公的機関へ観察事実、変更、危険、証跡を引き渡す | 受領者と指揮移管が明確 | 指示を復唱し区域維持 | 引渡し先、時刻 |
| 9 | 施設責任者へ利用制限と残留リスクを渡す | 再開判断者、条件が明確 | 判断まで区域・監視継続 | 指示、判断者 |
| 10 | 事件・事故案件を登録し次勤務へ引き継ぐ | 担当、期限、証跡、受領が明確 | 所有者不明は異常上申 | 案件ID、受領 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 要注意 | 不自然だが直ちに危険を確認できない | 距離確保、再確認、重点監視 | 条件付き可 |
| 不審 | 無断立入、説明不一致、所有者不明物等 | 応援、区域、責任者・警察連絡 | 引渡し後条件付き可 |
| 危険 | 暴力、凶器、爆発物兆候、危険物漏えい等 | 退避、緊急通報、避難支援 | 不可 |
| 指揮移管済み | 公的機関等が受領し指揮が明確 | 支援、区域・証跡維持 | 初動は条件付き可 |
| 未解決 | 対象見失い、所有者不明、危険未排除 | 監視・制限・案件継続 | 案件は未完了 |

## 10. 異常時対応

- 対象の退出、警報復帰又は物品の撤去だけで案件完了としない。
- 威嚇、暴力、凶器又は爆発物の疑いでは単独対応を中止し、退避・通報する。
- 不審物を安全確認のために触れる、移動する、開封する行為を行わない。
- 対象を見失った場合は最終位置・方向・特徴を共有し、危険な捜索・追跡をしない。
- 公的機関指示と施設運用判断を区別し、区域解除・利用再開は権限者が決定する。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 観察、退避、応援要請、区域設定、記録 | L0〜L1又は緊急権限 |
| 安全条件下の声掛け・立入保留 | 警備計画で認められたL1 |
| 緊急通報 | 物件別基準。切迫時は承認前に実施 |
| 身体拘束、所持品検査、不審物処理 | 標準手順の対象外 |
| 区域解除、施設利用再開 | 公的機関・施設責任者 |
| 対外説明、証跡開示、重大リスク判断 | 指定権限者・L3〜L4 |

## 12. 記録・完了条件

- [ ] 認知、観察、誘導、連絡、到着、引渡しを時系列で記録した
- [ ] 観察事実、申告及び推定を区別した
- [ ] 対象の特徴・位置・行動と、現場変更を記録した
- [ ] 区域、利用者動線、残留危険及び解除判断者を記録した
- [ ] 映像・ログ・物品等の証跡ID、保全者及び引渡し先を記録した
- [ ] 公的機関・施設責任者への指揮・対応責任の移管を確認した
- [ ] 未解決事項、重点監視、担当・期限を次勤務へ引き継いだ

対象の退出・撤去だけでは完了しない。身体安全、現場、証跡及び後続案件の対応責任が受領されていることを完了条件とする。
