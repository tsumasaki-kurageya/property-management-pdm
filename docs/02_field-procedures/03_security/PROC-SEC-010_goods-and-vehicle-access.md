# PROC-SEC-010 物品・車両出入管理

## 0. 文書管理

| 項目 | 内容 |
|---|---|
| 手順ID | `PROC-SEC-010` |
| 手順名 | 物品・車両出入管理 |
| 領域 | 警備 |
| 版 | 0.1 |
| 状態 | 初版草案 |
| 作成日 | 2026-07-25 |

> 個別物件の搬入出規則、車両動線、搬出承認、危険物、検査権限、個人情報及び緊急運用を優先する。

## 1. 目的

物品の搬入・搬出及び車両の入構・退構について、対象、申請、承認、数量、時間、経路及び受入先を照合する。人の入退館とは異なる搬出権限、車両安全及び受渡し証跡を確定する。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-02` | 人・物品・車両の出入を管理する | 物品・車両の許可と出入実績を管理する |
| 関連業務 | `BM-11-03` | 鍵を管理する | 搬入口等の鍵・媒体管理へ接続する |
| 関連業務 | `BM-12-01`〜`05` | 受付・確認・割当 | 受入先と例外確認へ接続する |
| 関連業務 | `BM-13-02` | 作業結果を記録する | 入退構、搬入出、承認を記録する |

## 3. 適用範囲

- 納品、回収、工事資材、備品等の搬入・搬出
- 配送車、工事車両、業務車両等の入構・退構
- 申請・承認、車両・運転者、物品、数量、経路及び時間の照合

法的根拠又は規程上の権限がない荷物・車内の検査、危険物の専門判定及び交通法規上の公的規制は対象外とする。

## 4. 開始条件

- [ ] 搬入出規則、申請、承認者、受入先、許可時間・経路を確認した
- [ ] 車両待機場所、歩行者分離、高さ・重量等の施設制約を確認した
- [ ] 搬出時の所有・持出承認と数量照合方法を確認した
- [ ] 危険物、廃棄物、時間外、大量搬入等の例外経路を確認した

## 5. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 運転者、車両、目的、受入先を確認する | 申請情報と一致 | 不一致は待機・確認 | 到着、確認結果 |
| 2 | 搬入出物品、数量、搬出承認を照合する | 対象と権限が明確 | 推測で許可しない | 伝票、承認ID |
| 3 | 入構条件、経路、速度、待機・荷役場所を案内する | 歩車分離と施設制約を満たす | 混雑・危険時は保留 | 条件、案内 |
| 4 | 受入先へ到着・受領を引き渡す | 受領者が明確 | 未受領は指定場所で待機 | 受領者、時刻 |
| 5 | 退構時に車両、貸与物、搬出内容を照合する | 未承認持出・未返却がない | 退構保留、責任者確認 | 退構、差異 |
| 6 | 例外・未解決を案件として引き継ぐ | 担当、期限、監視条件が明確 | 異常上申へ | 案件ID、受領 |

## 6. 判定・異常時対応

| 状態 | 基準 | 対応 |
|---|---|---|
| 許可 | 申請、承認、対象、時間、経路が一致 | 条件を案内し記録 |
| 条件付き許可 | 誘導、時間変更、範囲縮小等を承認者が指定 | 条件と承認者を記録 |
| 保留・拒否 | 情報不足、不一致、未承認、施設条件不適合 | 入構・搬出させず確認 |
| 危険 | 暴走、接触、漏えい、危険物疑い等 | 退避、区域管理、緊急連絡 |

事故は`PROC-SEC-007`、不審事象は`PROC-SEC-006`へ接続する。警備担当者が所有権や危険物適合を独自判断しない。

## 7. 権限・完了条件

- L0：規則どおりの照合、許可、案内、記録
- L1：安全確保のための一時停止、待機、再照合
- L2以上：時間外・申請外の例外、動線変更、広範な利用制限

- [ ] 入退構、搬入出対象、申請・承認、受入・搬出先を記録した
- [ ] 貸与物と未解決の差異を追跡できる
- [ ] 受入先への引渡し又は例外案件の受領が成立した

ゲート通過だけでは完了しない。対象物・車両と承認、実績、受領先が対応し、未解決事項の責任が確定した状態を完了とする。
