# PROC-SEC-008 火災・地震等の災害初動

> 本文書は警備担当の標準初動である。消防計画、防災計画、自衛消防組織、避難確保計画、自治体情報及び消防・警察等の指示を優先し、それらを置き換えない。

## 1. 目的

火災、地震、風水害等の災害を認知した際に、人命を最優先として計画を起動し、通報、情報伝達、初期消火、避難支援、区域管理及び設備担当との連携を行う。点呼・未確認者、危険区域、施設状態及び対応責任を災害指揮者・公的機関へ引き渡し、初動完了と災害収束・施設利用再開を区別する。

| 関連業務 | `BM-11-07`〜`BM-11-08` | 防災設備・訓練 | 設備確認と計画上の役割へ接続する |

## 3. 適用範囲

- 火災、煙、焦げ臭、爆発等の認知
- 地震発生時の身の安全、被害確認、避難・待機支援
- 台風、豪雨、浸水、高潮、強風、落雷、降雪等の初動
- 通報、館内伝達、初期消火、避難支援、点呼、区域管理及び指揮移管

災害原因の専門判断、建物の構造安全判定、消防・救助活動、復旧工事及び事業継続・施設再開の最終判断は対象外とする。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 第一認知者・警備員 | 身の安全、発信、通報、初動、観察・記録 |
| 防災センター・監視担当 | 情報集約、館内放送、設備状態、連絡統制 |
| 警備責任者・自衛消防担当 | 配置、避難・初期消火、点呼、現場統制 |
| 設備担当 | 防災設備、停止・隔離、技術情報、復旧確認 |
| 施設責任者・災害対策本部 | 施設運用、避難・待機、事業継続、再開判断 |
| 消防・警察・自治体 | 公的な消火、救助、避難指示、現場指揮 |

## 5. 開始条件

緊急時は書面・承認を待たず、身の安全、発信、119等への通報及び避難行動を優先する。

- [ ] 災害種別、場所、発生・認知時刻及び情報源を確認した
- [ ] 火煙、倒壊、落下物、浸水、感電、ガス等の切迫危険を確認した
- [ ] 消防計画・防災計画上の役割、指揮者、集合場所を確認した
- [ ] 通報、館内放送、無線、非常電源等の利用可否を確認した
- [ ] 避難経路、待機場所、要配慮者支援及び立入禁止区域を確認した

## 6. 必要資料・装備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 計画 | 消防・防災計画、役割表、避難図、連絡網 | 最新版、指揮系統、代替者 |
| 通信 | 電話、無線、館内放送、非常通報 | 停電・輻輳時の代替 |
| 防災 | 消火器、誘導灯、ヘルメット、ライト、保護具 | 使用可能性、訓練範囲 |
| 点呼 | 在館・来訪・勤務情報、点呼票 | 未確認者と最終確認位置 |
| 区域 | コーン、バー、案内表示 | 消防・避難動線を妨げない |

## 7. 災害別の初動原則

| 事象 | 初動原則 |
|---|---|
| 火災 | 発信・119・避難を優先し、初期消火は退路と安全が確保できる訓練範囲のみ |
| 地震 | 揺れの最中は身の安全を確保し、収束後に火気・負傷・設備・経路を確認 |
| 風水害・浸水 | 地下・低所・水際へ不用意に接近せず、早期の立入規制・上階避難を計画に従い実施 |
| 強風・落雷・降雪 | 屋外・屋上・ガラス・飛来物・電気設備への接近を避ける |
| 複合災害 | 一つの原因に限定せず、火災、停電、漏水、通信断等を並行して把握 |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 身の安全を確保し、災害種別・場所・危険を認知する | 接近・初動の安全性を判断 | 危険区域から退避 | 時刻、場所、事実 |
| 2 | 計画に従い発信し、119等と施設指揮者へ連絡する | 住所、進入口、危険、人数を伝達 | 通信不能は代替手段 | 発信・受領時刻 |
| 3 | 館内放送・誘導で避難又は安全な待機を案内する | 災害別計画と公的指示に一致 | 放送不能は配置・代替伝達 | 内容、対象、時刻 |
| 4 | 安全・訓練範囲で初期消火又は被害拡大防止を行う | 退路、装備、中止条件が成立 | 危険なら直ちに退避 | 実施者、結果 |
| 5 | 避難経路、消防進入路、危険区域を管理する | 交差・滞留・再進入を防止 | 代替経路・区域拡大 | 区域、変更 |
| 6 | 要配慮者、負傷者、取り残され情報を集約する | 無理な救助で二次被害を生じさせない | 消防等へ位置情報を渡す | 人数、最終位置 |
| 7 | 設備担当と防災設備・停止・隔離状態を共有する | 操作権限と影響が明確 | 無承認操作をしない | 警報、停止、指示 |
| 8 | 集合場所等で点呼し未確認者を整理する | 在館情報との差異を明示 | 危険区域へ捜索に戻らない | 確認・未確認者 |
| 9 | 公的機関・災害指揮者へ現場、点呼、危険、設備を引き渡す | 受領者と指揮移管が明確 | 指示を復唱し支援継続 | 引渡し先、時刻 |
| 10 | 初動結果、利用制限、残留危険、継続案件を引き継ぐ | 担当、期限、次回確認が明確 | 受領まで区域・監視継続 | 案件ID、受領 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 初動継続 | 通報・避難・消火・点呼等を実施中 | 指揮下で継続 | 不可 |
| 指揮移管済み | 公的機関・災害指揮者が受領 | 支援、情報提供、区域維持 | 初動は条件付き可 |
| 危険継続 | 火煙、倒壊、浸水、設備危険、未確認者等が残る | 立入禁止・案件継続 | 災害案件は未完了 |
| 初動完了 | 計画上の初動と引渡しが成立 | 記録・継続管理へ | 可 |
| 再開判断待ち | 災害影響は収束傾向だが安全確認未完了 | 利用制限を維持 | 再開不可 |

## 10. 異常時対応

- 火災・倒壊・浸水等の危険区域へ、捜索や確認のため単独で戻らない。
- 初期消火は火勢、煙、退路又は装備の条件を満たさなくなった時点で中止する。
- 館内放送・通信・非常電源が使えない場合は、代替伝達、配置及び応援を設定する。
- 在館者名簿と点呼結果の不一致を「避難済み」と推定せず、未確認として引き渡す。
- 警報復帰、鎮火情報、揺れの収束だけで安全・施設再開と判断しない。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 発信、退避、緊急通報、計画上の避難誘導 | L0〜L1又は緊急権限 |
| 訓練範囲の初期消火、区域設定、点呼 | 計画で指定された担当 |
| 設備停止・隔離・防災設備操作 | 指定担当・緊急権限 |
| 全館避難、閉鎖、帰宅・待機方針 | 災害指揮者・施設責任者・公的機関 |
| 危険区域解除、建物安全判定、施設利用再開 | 公的機関・有資格者・施設責任者 |
| 対外説明、事業継続、重大リスク受容 | L3〜L4 |

## 12. 記録・完了条件

- [ ] 発生・認知、発信、通報、放送、避難、到着、引渡し時刻を記録した
- [ ] 観察事実、公的情報、申告及び推定を区別した
- [ ] 初期消火・被害防止の実施者、条件、中止・結果を記録した
- [ ] 避難・待機場所、要配慮者、確認者・未確認者を記録した
- [ ] 危険区域、防災設備、停止・隔離、通信不能と代替措置を記録した
- [ ] 公的機関・災害指揮者への指揮移管を確認した
- [ ] 残留危険、利用制限、継続案件の担当・期限・受領を確認した
- [ ] 初動完了と災害収束・施設利用再開を別々に記録した

初動行動を実施しただけでは完了しない。計画上の役割を果たし、点呼・危険・設備・区域・継続案件が災害指揮者等に受領されていることを初動の完了条件とする。
