> 本文書は標準手順である。個別物件の緊急計画、救命講習、AED運用、消防・警察・救急への通報基準、個人情報、事故報告及び現場保全規程を優先する。

## 1. 目的

事故、負傷、急病等を認知した際に、警備員自身と周囲の安全を確保し、人命を最優先に通報・救護・誘導を行う。事実を時系列で記録し、救急・警察・施設責任者等へ対応責任、現場及び証跡を適切に引き渡す。搬送又は救護終了と、事故案件・現場保全の完了を区別する。

## 2. 業務カタログとの関係

## 3. 適用範囲

- 転倒、衝突、落下、挟まれ等の事故
- 急病、意識障害、負傷者の発見
- 利用者・作業者からの救護要請
- 二次災害防止、通報、救急隊等の誘導、現場保全及び引渡し

火災、地震、暴力・凶器、危険物、設備災害等は、それぞれの専門・災害手順を優先し、本手順を必要な範囲で併用する。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 第一認知者・警備員 | 安全確認、通報、訓練範囲の救護、連絡、記録 |
| 警備責任者 | 応援配置、動線・区域、施設連絡、記録統制 |
| 救命教育修了者等 | 受講内容と物件計画の範囲で救命処置 |
| 施設責任者 | 施設運用、利用制限、関係者対応、案件管理 |
| 救急・消防・警察 | 公的な救護、捜査、現場指揮 |
| 設備担当 | 設備停止・安全化・技術確認 |

## 5. 開始条件

緊急時は書面確認を待たず、人命保護と緊急通報を優先する。可能な範囲で次を確認する。

- [ ] 場所、事象、負傷・急病者数及び認知時刻
- [ ] 火災、感電、漏水、落下物、暴力等の二次危険
- [ ] 自身の退路、応援、通信及び救急隊の進入経路
- [ ] AED、救急用品、担架等の位置と使用可能性
- [ ] 施設責任者、救急・警察等への連絡方法

## 6. 必要資料・装備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 連絡 | 緊急連絡網、住所・進入口、施設情報 | 通報時に伝える内容 |
| 救護 | AED、救急用品、感染防護具 | 点検、有効期限、教育範囲 |
| 区域 | コーン、バー、案内表示 | 救急動線と避難経路を妨げない |
| 記録 | 時計、記録票、無線、必要時カメラ | 個人情報、撮影・開示権限 |

## 7. 初動時の危険予知

| 危険 | 原則 |
|---|---|
| 感電、火災、ガス、倒壊、交通 | 安全化前に接近せず、専門機関へ通報 |
| 暴力・凶器・不審者 | 距離・遮蔽物・退路を確保し警察・応援へ |
| 血液・体液 | 感染防護具を使用し直接接触を避ける |
| 群集・撮影・個人情報 | 救急動線を確保し、不要な撮影・情報共有を抑止 |
| 証拠・遺留品 | 救命に必要な場合を除き移動・清掃・廃棄しない |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 安全な位置から場所、人数、状態、二次危険を確認する | 接近・救護の安全性を判断 | 危険なら接近せず通報 | 認知時刻、観察事実 |
| 2 | 必要に応じ119・110、施設責任者、応援へ連絡する | 場所、状態、危険、進入口を伝達 | 通信不能は代替手段 | 発信・受領時刻 |
| 3 | 訓練・指令の範囲で反応・呼吸等を確認し救護する | 救命手順と指令に従う | 無理な移動・医療判断をしない | 処置、開始時刻 |
| 4 | AED・救急用品を手配し、救急隊の進入経路を確保する | 装備と誘導担当が配置済み | 代替経路・応援を設定 | 装備、担当、到着 |
| 5 | 周囲を区域管理し二次災害・群集を防ぐ | 救護・避難動線を妨げない | 区域を拡大・施設利用制限へ | 区域、変更時刻 |
| 6 | 関係者の申告と観察事実を区別して時系列記録する | 推定・診断を断定しない | 情報源不明は明記 | 発言者、時刻、原文 |
| 7 | 救急・警察等へ状態、処置、危険、関係情報を引き渡す | 受領者と指揮移管が明確 | 追加確認に対応 | 引渡し先、時刻 |
| 8 | 現場、物品、映像等を必要な範囲で保全する | 救命を妨げず改変を抑える | 清掃・復旧を保留 | 証跡ID、管理者 |
| 9 | 施設責任者へ運用影響、残留危険、利用可否の判断材料を渡す | 救護と施設利用再開を分離 | 権限者不在は区域維持 | 判断者、指示 |
| 10 | 事故案件を登録し、報告・調査・引継ぎへ接続する | 担当、期限、受領が明確 | 所有者不明は上申 | 案件ID、受領 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 安全確認済み | 二次危険を管理し救護・誘導できる | 救護・通報継続 | 初動継続 |
| 危険継続 | 火災、感電、暴力、倒壊等が未管理 | 接近回避、退避、専門機関へ | 不可 |
| 引渡し済み | 公的機関等が受領し指揮・責任が明確 | 支援・現場保全 | 条件付き可 |
| 現場保全継続 | 搬送後も危険、証跡、調査、利用制限が残る | 区域維持・案件追跡 | 案件は未完了 |

## 10. 異常時対応

- 生命危険又は判断に迷う状態では、施設内承認を待たず緊急通報する。
- 暴力・凶器等は単独で制止・追跡せず、退避・警察連絡・利用者誘導を行う。
- 感電、火災、ガス、倒壊等では設備・消防等による安全化前に接近しない。
- 死亡の断定、病名・原因の診断、個人情報の不要な開示を行わない。
- 救命のため現場や物品を動かした場合は、可能な範囲で変更前後と理由を記録する。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 認知、通報、応援要請、区域設定、記録 | L0〜L1又は緊急権限 |
| 教育・指令範囲の救命処置 | 教育・物件計画による |
| 施設閉鎖、営業中止、利用制限 | L2以上 |
| 個人情報・映像の外部開示 | 指定権限者 |
| 現場解除・利用再開 | 施設責任者又は公的機関 |
| 対外説明、補償、重大リスク判断 | L3〜L4 |

## 12. 記録・完了条件

- [ ] 認知から通報、処置、到着、引渡しまでの時刻を記録した
- [ ] 観察事実、関係者の申告、作業者の判断を区別した
- [ ] 二次危険、区域、救急・避難動線及び変更を記録した
- [ ] 救護・AED等の実施者、内容、時刻を記録した
- [ ] 公的機関、施設責任者への引渡し先・時刻・指示を記録した
- [ ] 現場・物品・映像等の保全又は変更理由を記録した
- [ ] 残留危険、利用制限、事故案件の担当・期限・受領を設定した
- [ ] 次勤務へ未解決事項を引き継いだ

搬送又は応急救護の終了だけでは完了しない。対応責任、現場、証跡、残課題が受領されていることを完了条件とする。
