> 本文書は標準手順である。個別物件の警備計画、勤務基準、鍵・装備台帳、個人情報取扱い、緊急連絡及び指揮命令系統を優先する。

## 1. 目的

勤務開始・交代時に、人員、配置、装備、鍵、警報、継続案件、工事、残留者、臨時指示等を照合し、警備機能の空白や情報欠落を防ぐ。未解決事項について受領者、次の確認時刻及び上申条件を明確にし、勤務の連続性を確保する。

## 2. 業務カタログとの関係

警備勤務の交代・引継ぎ自体に直接対応する業務IDは、現行カタログでは未設定である。本手順を試作し、独立業務として追加する必要性を検証する。

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | 未設定 | 警備勤務の開始・交代（不足候補） | 勤務機能と継続案件を受け渡す |
| 関連業務 | `BM-05-04` | シフトを作成する | 予定配置と実配置を照合する |
| 関連業務 | `BM-11-01` | 警備計画を作成する | 配置、巡回、重点箇所を確認する |
| 関連業務 | `BM-11-10` | 警備・防災記録を報告する | 前勤務の結果と継続事項を引き継ぐ |
| 関連業務 | `BM-13-01`〜`BM-13-02` | 作業開始・内容を記録する | 開始、配置、受領事項を記録する |
| 関連業務 | `BM-17-06` | 危険予知・リスク評価を行う | 当勤務の危険・対応条件を確認する |

## 3. 適用範囲

### 3.1 適用対象

- 常駐警備の勤務開始、対面交代及び無人時間帯を挟む引継ぎ
- 巡回・機械警備の担当開始、遠隔引継ぎ及び駆付け待機開始
- 鍵、カード、無線、端末、防災・救護用品等の装備受渡し
- 継続案件、警報、事故、工事、残留者、要注意区画、臨時指示の受渡し

### 3.2 適用除外・使用禁止条件

- 資格・教育・配置条件を満たさない者への勤務引渡し
- 重大な人員不足、通信不能又は必須装備欠落を未報告のまま勤務開始
- 個人情報・機密情報を許可されていない媒体・相手へ伝達
- 鍵・装備の数量不一致を後回しにした形式的な受領
- 緊急事象の対応中に指揮・現場を無秩序に交代すること

## 4. 実施体制

| 役割 | 主な責任 | 必要な力量 |
|---|---|---|
| 引継ぎ者 | 前勤務の事実、継続案件、装備を整理して説明する | 物件教育、記録方法、守秘 |
| 受領者 | 復唱・現物照合し、不明・不足を開始前に確認する | 物件教育、配置に必要な力量 |
| 警備責任者 | 配置、人員不足、未解決事項、例外開始を判断する | 警備計画・契約・指揮系統の理解 |
| 施設責任者 | 運用変更、工事、利用制限等を共有・決定する | 物件別権限 |
| 設備・防災担当 | 設備停止、警報、火気・防災上の注意を共有する | 対象専門性 |

## 5. 開始条件

- [ ] 当日の警備計画、シフト、配置、担当時間及び休憩交代を確認した
- [ ] 引継ぎ者と受領者又は承認済みの代替引継ぎ手段を確保した
- [ ] 勤務開始前に記録・装備・鍵を照合する時間を確保した
- [ ] 指揮命令、定時連絡、応援、警察・消防・救急への接続方法を確認した
- [ ] 受領者の体調、資格・教育、単独勤務条件を確認した
- [ ] 未解決の緊急事象がある場合、指揮権移管方法を責任者が決定した

## 6. 引継ぎ対象

| 区分 | 主な内容 | 確認方法 |
|---|---|---|
| 人員・配置 | 出欠、持場、休憩、応援、欠員 | シフトと実員の照合 |
| 鍵・装備 | 鍵、カード、無線、ライト、端末、救護・防災用品 | 現物、番号、数量、動作 |
| 警報・設備 | 発報、停止、故障、監視除外、工事影響 | 監視画面、日誌、担当者情報 |
| 継続案件 | 事故、不審者、遺失物、未返却、区域閉鎖、要観察 | 案件ID、状態、次回条件 |
| 人・利用 | 残留者、来訪予定、要注意情報、催事、搬入 | 必要最小限の承認済み情報 |
| 工事・作業 | 作業者、区域、火気、停電、鍵、終了予定 | 作業許可・施設情報 |
| 臨時指示 | 巡回変更、重点監視、入館制限、連絡先役割 | 発信者、承認、期限 |
| 記録 | 前勤務の日誌、巡回、警報、連絡、未完了 | 抜け・矛盾・訂正履歴 |

## 7. 作業前確認・危険予知

| 危険・影響要因 | 予防措置 | 発生時の初動 |
|---|---|---|
| 情報欠落 | 口頭だけでなく記録・現物と照合する | 開始保留又は条件付き開始を上申 |
| 鍵・装備不一致 | 番号・数量・状態を双方で照合する | 利用停止、紛失初動、責任者連絡 |
| 人員不足・体調不良 | 必須配置・単独勤務条件を確認する | 応援、配置変更、勤務開始保留 |
| 緊急事象中の交代 | 指揮者、現場、通信、記録を一つずつ移管 | 公的機関・現指揮者の指示を優先 |
| 個人情報漏えい | 必要性、受領権限、媒体、保管を確認する | 伝達中止、責任者へ確認 |
| 無人時間・遠隔引継ぎ | 時刻同期、受領確認、代替連絡を設定する | 未受領なら勤務開始と見なさない |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 引継ぎ者が前勤務の記録と継続案件を整理する | 未記入、矛盾、期限超過がない | 訂正理由を残し責任者へ | 日誌、案件一覧 |
| 2 | 受領者の出勤、体調、資格・教育、配置を確認する | 必須配置と単独勤務条件を満たす | 応援・配置変更を上申 | 実員、配置、判断 |
| 3 | 鍵、カード、無線、端末、指定装備を双方で照合する | 番号、数量、動作、破損が一致 | 未返却・紛失・故障初動へ | 台帳、時刻、双方確認 |
| 4 | 警報、監視除外、設備停止、工事、火気等を共有する | 現在状態と注意・禁止事項が明確 | 設備・施設担当へ確認 | 状態、期限、担当 |
| 5 | 継続案件を一件ずつ説明し受領者が復唱する | 現在状態、次の行動・時刻、上申条件が明確 | 所有者不明は責任者へ | 案件ID、受領者 |
| 6 | 残留者、来訪・搬入、重点監視、臨時指示を共有する | 必要最小限で有効期限・根拠がある | 不明・期限切れは適用保留 | 指示元、期限 |
| 7 | 当勤務の巡回、受付、監視、休憩、定時連絡を確認する | 担当と代替要員が重複・空白なく定まる | 責任者が再配置 | 配置表、予定 |
| 8 | 受領判定を行い、双方が記録を確定する | 開始可否と未解決事項が明確 | 9章の区分で処理 | 署名・電子承認、時刻 |
| 9 | 勤務開始を記録し、必要先へ交代完了を連絡する | 指揮・通信・監視が継続している | 旧勤務者を解放せず責任者へ | 開始・終了、連絡 |
| 10 | 引継ぎ後早期に重要点・監視画面を照合する | 口頭情報と現場・画面が一致 | 相違を新規案件化 | 初回確認結果 |

## 9. 受領判定

| 状態 | 条件 | 標準対応 |
|---|---|---|
| 開始可 | 人員・情報・装備・指揮が揃い、未解決事項に担当・期限がある | 受領し勤務開始 |
| 条件付き開始 | 軽微な不足があるが代替策、期限、責任者承認がある | 条件を記録して開始 |
| 開始保留 | 重要情報不足、鍵・装備不一致、人員条件未達等がある | 前勤務・責任者と是正 |
| 開始不可 | 法定・契約配置未達、通信不能、重大安全問題、指揮不明 | 勤務開始せず緊急上申 |

引継ぎ者の勤務終了と、受領者の勤務開始は自動的に同時成立するものではない。受領完了を確認する。

## 10. 異常時対応

| 事象 | 初動 | 後続 |
|---|---|---|
| 無断欠勤・人員不足 | 必須配置と業務優先度を確認し応援要請 | 責任者が配置・業務縮小を決定 |
| 鍵・カード不一致 | 対象を特定し権限停止・捜索範囲を統制 | `PROC-SEC-003`、施設責任者へ |
| 装備故障 | 代替品と配置条件を確認 | 代替不能なら開始保留 |
| 記録欠落・矛盾 | 事実確認し推測で補完しない | 訂正者、時刻、理由を保存 |
| 緊急事象継続 | 指揮・通信・現場・記録を明示的に移管 | 公的機関又は現指揮者の承認を得る |
| 個人情報の誤伝達 | 伝達を停止し範囲を保全 | 責任者・情報管理手順へ接続 |

## 11. 権限・上申

| 判断・行為 | 権限 | 標準的な扱い |
|---|---|---|
| 計画どおりの照合・受領・記録 | L0 | 引継ぎ者・受領者が実施 |
| 軽微な装備交換、確認追加 | L1 | 承認済み範囲で実施 |
| 配置変更、巡回変更、条件付き開始 | L2 | 警備責任者等が承認 |
| 追加要員・契約外対応 | L3 | 契約・予算権限者が決定 |
| 恒常的な警備計画・リスク受容変更 | L4 | オーナー等が決定 |
| 緊急通報・退避 | 緊急権限 | 切迫時は引継ぎより優先 |
| 指揮権移管 | 現指揮者・上位責任者 | 受領確認を明示する |

## 12. 記録・完了条件

- [ ] 引継ぎ者・受領者、勤務開始・終了時刻、配置を記録した
- [ ] 鍵・カード・装備の番号、数量、状態及び双方確認を記録した
- [ ] 継続案件ごとに状態、担当、次回時刻、上申条件を設定した
- [ ] 警報、停止、監視除外、工事、残留者、臨時指示を記録した
- [ ] 不足・相違、条件付き開始の代替策、期限、承認者を記録した
- [ ] 受領確認後に前勤務者を解放し、監視・指揮の空白がないことを確認した
