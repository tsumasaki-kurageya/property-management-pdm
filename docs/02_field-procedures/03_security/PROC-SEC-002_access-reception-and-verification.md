# PROC-SEC-002 入退館受付・確認

## 0. 文書管理

| 項目 | 内容 |

| 手順ID | `PROC-SEC-002` |
| 手順名 | 入退館受付・確認 |
| 領域 | 警備 |
| 版 | 0.1 |
| 状態 | 初版草案 |

> 本文書は標準手順である。個別物件の入退館規則、本人確認方法、予約・承認経路、制限区域、個人情報、入館証・鍵管理及び緊急時運用を優先する。

## 1. 目的

来訪者・作業者等について、本人、訪問先、目的、時間帯及び立入可能範囲を承認済み情報と照合し、入館許可、条件付き許可、確認保留又は拒否を一貫して判断する。受付処理と実際の退館確認を追跡し、例外を推測や慣例で許可しない。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-02` | 入退館を管理する | 受付、照合、許可、退館確認を行う |
| 関連業務 | `BM-12-01`〜`BM-12-05` | 受付・確認・割当 | 訪問先確認と例外連絡へ接続する |
| 関連業務 | `BM-13-02` | 作業結果を記録する | 受付・許可・退館の事実を残す |
| 関連業務 | `BM-13-11` | 異常を速報・上申する | 不一致・無断立入等を上申する |
| 関連業務 | `BM-17-07` | 事故・ヒヤリハットを管理する | 重大な規則違反・事案へ接続する |

## 3. 適用範囲

- 来訪者、納品者、工事・保守作業者等の受付
- 予約、本人、訪問先、目的、時間帯、立入範囲の照合
- 入館可否判断、必要な入館証等の発行依頼・記録
- 退館、入館証等の返却、未退館・未返却の例外処理

従業員の恒常的な権限付与、鍵・入館証の在庫・貸出管理、所持品検査、身体拘束等は対象外とし、`PROC-SEC-003`又は物件別規程へ接続する。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 受付・警備担当者 | 本人・予約・目的・範囲の照合、記録、案内 |
| 訪問先・受入担当者 | 来訪目的、受入可否、同行・範囲、時間の承認 |
| 警備責任者 | 不一致、拒否、規則違反、例外承認の統制 |
| 施設責任者 | 入退館規則、制限区域、運用例外、閉鎖等の判断 |
| システム・証票管理者 | 予約・権限情報、入館証等の有効性管理 |

## 5. 開始条件

- [ ] 最新の入退館規則、受付対象、本人確認方法を確認した
- [ ] 予約・承認情報、訪問先連絡先、制限区域を確認できる
- [ ] 受付記録、入館証等、返却管理の手段が利用できる
- [ ] 確認不能・拒否・不審・緊急時の連絡経路を確認した
- [ ] 個人情報の閲覧・記録・開示範囲を確認した

## 6. 必要資料・設備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 規則 | 入退館規則、本人確認基準、制限区域 | 最新版、時間帯別条件 |
| 予約 | 来訪予定、作業届、受入承認 | 氏名、会社、目的、範囲 |
| 証票 | 入館証、来訪者証、返却台帳 | 有効期限、番号、在庫 |
| 連絡 | 訪問先、警備責任者、施設責任者 | 代替連絡、時間外経路 |
| 記録 | 受付簿、端末、監査記録 | 必要最小限、保存・閲覧権限 |

## 7. 受付時の安全・情報管理

- 証明書等は必要な範囲だけ確認し、不要な複写・撮影を行わない。
- 予約情報・在館者情報・制限区域・鍵情報を第三者へ不用意に開示しない。
- 判断に迷う相手へ、確認質問の正答や内部情報を先に示さない。
- 威圧、暴力、無断通過等のおそれがある場合は距離・退路・応援を確保する。
- システム障害時も、本人確認・受入確認を省略せず代替手順を用いる。

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 来訪目的と訪問先を聴取する | 対象施設・受付区分に該当 | 対象外は所定窓口へ | 到着時刻、申告 |
| 2 | 承認済み方法で本人・所属を確認する | 本人情報が一致 | 不一致・確認不能は保留 | 確認方法、結果 |
| 3 | 予約・作業届・受入承認を照合する | 日時、人数、目的が一致 | 訪問先へ確認 | 予約ID、差異 |
| 4 | 訪問先へ受入可否を確認する | 承認者と回答が明確 | 未応答時は入館させない | 相手、時刻、回答 |
| 5 | 立入範囲、経路、同行、禁止事項、退出時刻を設定する | 必要最小限の権限 | 例外は責任者承認 | 範囲、条件 |
| 6 | 入館証等を識別番号とともに交付・案内する | 交付物と本人が対応 | 不足・不良は代替又は保留 | 証票番号、時刻 |
| 7 | 入館を記録し必要な受入担当へ引き渡す | 受付から受入へ責任が移る | 受領未確認は待機 | 受領者、時刻 |
| 8 | 退館時に本人、時刻、貸与物、必要な退出確認を照合する | 返却・退出が完了 | 未返却・未退館を追跡 | 退館、返却 |
| 9 | 不一致・例外・未解決を責任者へ引き渡す | 担当、期限、監視条件が明確 | 無断立入等は緊急対応 | 案件ID、受領 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 入館可否 |
|---|---|---|---|
| 許可 | 本人、予約・承認、目的、時間、範囲が一致 | 条件を案内し記録 | 可 |
| 条件付き許可 | 責任者が同行・範囲縮小等の条件を明示し受領 | 条件と承認者を記録 | 条件付き可 |
| 確認保留 | 情報不足、未応答、システム障害等で確認未成立 | 安全な場所で待機、再確認 | 不可 |
| 拒否 | 明確な不一致、期限切れ、規則違反、承認なし | 理由を告げ上申 | 不可 |
| 危険 | 暴力、凶器、強行通過等の生命・施設危険 | 接近回避、応援・警察連絡 | 不可 |

## 10. 異常時対応

- 予約があることだけで本人確認を省略せず、本人が確認できても承認範囲外へ通さない。
- 「いつも入っている」「急いでいる」等を例外許可の根拠にしない。
- 強行通過、なりすまし、改ざん証票、不審行動は`PROC-COM-004`へ接続する。
- 暴力・凶器等では単独で身体制止せず、退避、応援要請、警察連絡を優先する。
- 未退館・未返却は在館・貸与記録を照合し、`PROC-SEC-003`へ接続する。
- システム障害時は代替記録を用い、復旧後に重複・漏れを照合する。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 規則どおりの受付・照合・許可・記録 | L0 |
| 承認済み代替確認、待機、再照合 | L1 |
| 条件付き許可、時間外・予約外の例外 | L2又は指定承認者 |
| 施設閉鎖、広範な入館停止、警察対応方針 | L3 |
| 恒常的な規則・権限・リスク受容変更 | L4 |

緊急通報と安全確保は通常の例外承認を待たず、物件別の緊急権限で実施する。

## 12. 記録・完了条件

- [ ] 到着・入館・退館時刻、本人・所属、訪問先、目的を必要最小限で記録した
- [ ] 本人確認方法、予約・承認、許可範囲、条件及び承認者を記録した
- [ ] 入館証等の識別番号、交付・返却を記録した
- [ ] 確認保留、拒否、例外、不一致の事実と対応を記録した
- [ ] 未退館・未返却・規則違反の担当、期限、監視条件及び受領を確認した
- [ ] 代替記録を使用した場合は復旧後の照合責任を設定した

受付で入館させたことだけでは完了しない。許可範囲が受入先へ引き渡され、退館・返却又は例外案件の継続責任が追跡できることを完了条件とする。
