# PROC-SEC-005 警報・防犯監視

> 本文書は標準手順である。個別物件の警備計画、機械警備契約、監視設備操作、警報優先度、映像・個人情報、通報・駆付け基準及び緊急計画を優先する。

## 1. 目的

防犯カメラ、侵入・開閉・非常通報等の監視情報を継続確認し、警報の対象、時刻、場所、真正性及び緊急度を判定する。人命と監視員の安全を優先して現場確認、応援・公的機関への連絡、区域管理及び後続案件へ接続し、表示復帰、現場正常、対応責任の引渡し及び案件完了を区別する。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-05` | 監視設備を確認する | 警報・映像・センサーを監視する |
| 関連業務 | `BM-11-06` | 事故・事件へ対応する | 不審・侵入・非常通報へ接続する |
| 関連業務 | `BM-11-09` | 災害時対応を行う | 火災等の警報を専門計画へ渡す |
| 関連業務 | `BM-11-10` | 警備・防災記録を報告する | 受信・判断・連絡・引渡しを記録する |
| 関連業務 | `BM-13-01`〜`BM-13-05`、`BM-13-11` | 指示・記録・報告・異常上申 | 対応責任を引き渡す |

## 3. 適用範囲

- 防犯カメラ、侵入、扉開放、非常ボタン、動体・フェンス等の監視
- 警報受信時の情報照合、映像・関連点確認、緊急度判定
- 現場確認者への指示、応援・施設責任者・警察等への連絡
- 通信断、カメラ停止、死角等の監視不能時の代替監視

映像だけによる犯罪・本人の断定、権限外の追跡・制止、警報設備の修理、証跡の外部開示は対象外とする。

## 4. 実施体制

| 役割 | 主な責任 |
|---|---|
| 監視員 | 監視、受信、時刻記録、初期判定、連絡 |
| 現場確認者 | 安全な経路での現場確認、事実報告 |
| 警備責任者 | 優先度、配置、応援、対応方針、記録統制 |
| 施設責任者 | 利用制限、施設運用、外部対応の判断 |
| 機械警備会社 | 契約範囲の監視、駆付け、設備連携 |
| 警察・消防等 | 公的な初動・現場指揮 |

## 5. 開始・継続条件

- [ ] 監視対象、区画、警報種別、優先度及び連絡基準を確認した
- [ ] 監視端末、時刻、録画・通信、無線等が使用可能である
- [ ] 死角、停止中設備、工事、テスト、臨時解除を引き継いだ
- [ ] 現場確認者、応援、施設責任者及び公的機関への連絡方法を確認した
- [ ] 映像・個人情報の閲覧、保存、共有権限を確認した
- [ ] 単独接近を避ける事象と退避・緊急通報条件を確認した

## 6. 必要資料・設備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 監視 | 監視端末、警報一覧、区画図、カメラ配置 | 時刻、死角、状態 |
| 記録 | 警報ログ、勤務記録、端末 | 改ざん防止、同期 |
| 連絡 | 無線、電話、連絡網 | 代替手段、録音規程 |
| 現場 | 巡回経路、鍵、照明、保護具 | 安全経路、単独条件 |
| 証跡 | 映像保存・書出し手順 | 権限、保存期間、引渡し |

## 7. 初期判定

| 状態 | 代表条件 | 初動 |
|---|---|---|
| 危険・緊急 | 非常通報、暴力、凶器、侵入継続、火災等 | 退避・緊急通報・応援・利用者保護 |
| 異常 | 無断立入、扉開放継続、不審行動、複数警報 | 安全な現場確認、責任者連絡 |
| 要確認 | 単発警報、映像不鮮明、予定作業との不一致 | 関連情報・現場を追加確認 |
| 監視不能 | 通信断、録画停止、死角拡大、時刻不整合 | 代替監視、巡回強化、修繕上申 |
| 正常・試験 | 承認済みテスト又は予定作業と一致 | 記録し解除条件を確認 |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 警報原文、対象、発生・受信時刻を記録する | 種別・場所を識別 | 不明でも原文を保持 | コード、時刻 |
| 2 | 同時警報、関連カメラ、入退館・工事・解除予定を照合する | 正常予定／異常／不明を仮分類 | 推測で解除しない | 関連ログ |
| 3 | 映像・センサーの観察事実を確認する | 人物・行動・物品を客観記録 | 本人・意図を断定しない | 映像時刻、カメラID |
| 4 | 緊急度と現場確認可否を判定する | 人命、継続性、場所、応援を評価 | 危険時は現場接近させない | 判定、判断者 |
| 5 | 必要に応じ施設責任者、応援、警察・消防へ連絡する | 発信・受領・指示が明確 | 代替連絡を使用 | 相手、時刻、回答 |
| 6 | 現場確認者へ場所、経路、危険、通信、確認事項を復唱させる | 退路・応援を確保 | 単独危険時は出動保留 | 出動・到着時刻 |
| 7 | 安全な位置から現場を確認し監視員へ報告する | 映像・警報・現場の整合を判断 | 異常時は距離確保・区域管理 | 観察事実、報告 |
| 8 | 承認済み条件で警報を復帰・解除し、再発と関連点を確認する | 原因・現場・関連点が確認済み | 原因不明の反復解除は禁止 | 操作、再発 |
| 9 | 監視不能時は代替巡回・監視を設定する | 範囲、頻度、担当、期限が明確 | 安全担保不能は利用制限へ | 代替条件 |
| 10 | 結果、証跡、残課題を責任者・次勤務へ引き渡す | 受領と対応責任が明確 | 受領まで重点監視継続 | 案件ID、受領 |

## 9. 判定基準

| 状態 | 基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 正常確認 | 予定事象と一致し現場・関連点に異常なし | 記録し監視継続 | 可 |
| 要観察 | 直ちに危険なしだが原因未確定・反復・死角あり | 重点監視・期限付き引継ぎ | 条件付き可 |
| 異常 | 無断立入、不審行動、現場不一致、機能停止等 | 応援・初動・案件化 | 不可又は引渡し後条件付き可 |
| 危険 | 暴力、凶器、火災等の生命危険 | 退避・緊急通報・避難支援 | 不可 |
| 監視不能 | 必須監視を継続できない | 代替監視又は利用制限 | 条件付き又は不可 |

## 10. 異常時対応

- 危険が疑われる現場へ単独で接近・追跡・制止しない。
- 警報表示の消去や対象者の退出だけで案件完了としない。
- 不審者・不審物は`PROC-SEC-006`、事故・急病は`PROC-SEC-007`、災害は`PROC-SEC-008`へ接続する。
- 設備・通信不良は設備担当へ引き渡し、代替監視を維持する。
- 証跡は必要最小限を権限内で保全し、無断で複製・開示しない。

## 11. 権限・上申

| 行為 | 権限 |
|---|---|
| 監視、ログ照合、記録、承認済み試験確認 | L0 |
| 再確認、応援要請、立入保留、重点監視 | L1 |
| 警報解除、区域閉鎖、監視配置変更 | 物件別L1〜L2 |
| 警察・消防への緊急通報 | 物件別緊急基準。切迫時は承認前に実施 |
| 映像保存・書出し・外部開示 | 指定権限者 |
| 営業停止、長期利用制限、重大リスク受容 | L3〜L4 |

## 12. 記録・完了条件

- [ ] 警報原文、対象、発生・受信・確認・連絡・復帰時刻を記録した
- [ ] 映像・関連ログ・現場の観察事実と判断を区別した
- [ ] 緊急度、現場確認可否、応援・通報判断と根拠を記録した
- [ ] 警報操作、再発、監視不能、代替監視を記録した
- [ ] 証跡ID、保存・引渡し先、閲覧・開示状態を記録した
- [ ] 残課題、重点監視、担当・期限及び次勤務の受領を確認した

警報表示の復帰だけでは完了しない。現場又は代替手段で状態を確認し、異常・監視不能の対応責任と証跡が受領されていることを完了条件とする。
