# PROC-SEC-015 防災訓練の計画・実施・評価

> 個別物件の消防計画、防災計画、法令・行政指導、用途・在館者特性、関係機関との調整及び訓練中止基準を優先する。

## 1. 目的

訓練目的と評価項目を先に定め、関係者、シナリオ、安全統制、実施、観察、評価及び改善を一つのサイクルとして管理する。実災害との誤認や訓練中の事故を防ぎ、参加実績だけでなく初動体制の有効性と改善責任を確定する。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-08` | 防災訓練を計画・実施する | 企画、実施統制、評価、改善を行う |
| 関連業務 | `BM-11-07` | 防災設備を確認する | 訓練で使用する設備状態を事前確認する |
| 関連業務 | `BM-11-09` | 災害時対応を行う | 実災害初動手順を検証する |
| 関連業務 | `BM-11-10` | 警備・防災記録を報告する | 訓練結果と改善事項を報告する |

## 3. 適用範囲

- 火災、地震、風水害等を想定した机上・部分・総合訓練
- 通報、初期消火、避難誘導、点呼、情報伝達、指揮移管等の検証
- 目的設定、関係者調整、シナリオ、安全計画、実施、評価、改善管理

実災害対応、法定点検の代替及び参加者へ知らせない危険な演出は対象外とする。

## 4. 実施体制・開始条件

| 役割 | 主な責任 |
|---|---|
| 訓練責任者 | 目的、範囲、実施・中止、総括を承認する |
| 訓練統制者 | シナリオ進行、安全監視、実災害切替えを統制する |
| 各役割担当 | 消火、通報、誘導、点呼、設備等を計画どおり実施する |
| 評価者 | 行動、時刻、判断、連携を観察し評価する |
| 施設・関係機関 | 利用影響、設備、消防等との調整・指示を行う |

- [ ] 訓練目的、対象能力、評価項目及び合格・改善判断を定めた
- [ ] 消防計画等との整合、関係者・利用者への周知範囲を確認した
- [ ] 訓練区域、設備、模擬表示、通信、救護及び中止条件を確認した
- [ ] 「訓練」と実災害を識別し、実災害時に即時切り替える方法を確認した
- [ ] 要配慮者、業務継続、避難経路及び第三者への影響を確認した

## 5. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 過去課題と計画から目的・評価項目を定める | 検証対象が具体的 | 参加だけを目的にしない | 目的、評価表 |
| 2 | シナリオ、役割、時系列、想定付与を設計する | 実初動と対応する | 過度な演出を除く | 計画、役割表 |
| 3 | 関係者、設備、区域、周知、安全措置を調整する | 実施・中止権限が明確 | 条件未達は延期 | 承認、周知 |
| 4 | 事前説明と装備・通信・避難経路確認を行う | 参加者が訓練と安全条件を理解 | 不明者を配置しない | 参加、確認 |
| 5 | 統制者が訓練開始を宣言し想定を付与する | 実災害と識別可能 | 誤認時は停止・訂正 | 開始、想定 |
| 6 | 各担当が初動し、評価者が事実と時刻を記録する | 安全条件内で実施 | 危険・実災害時は中止 | 行動、時刻 |
| 7 | 終了宣言、設備・区域・人員の復帰を確認する | 通常状態へ安全に戻る | 規制を維持し是正 | 終了、復帰 |
| 8 | 振り返りで事実、原因、改善案を整理する | 個人批判でなく仕組みを評価 | 追加確認を設定 | 評価、課題 |
| 9 | 改善担当、期限、効果確認を決定し承認を得る | 課題が追跡可能 | 未受領は訓練未完了 | 改善台帳、受領 |

## 6. 判定・中止・実災害切替え

| 状態 | 基準 | 対応 |
|---|---|---|
| 実施可 | 安全、周知、設備、統制、救護条件を満たす | 計画どおり開始 |
| 条件付き | 一部項目を安全に除外・縮小できる | 条件を承認し記録 |
| 中止・延期 | 安全条件未達、設備不良、重大な運用競合 | 実施せず再計画 |
| 実災害発生 | 火災、地震、急病、事故等が実際に発生 | 訓練を中止し`PROC-SEC-008`等へ |

訓練統制者は危険兆候、実災害、通信不能又は第三者影響の拡大時に中止できる。中止後の施設利用再開は別権限者が判断する。

## 7. 権限・完了条件

- L0：承認済み計画内の役割実施、観察、記録
- L1：危険時の訓練中止、退避、救護・緊急通報
- L2以上：計画変更、区域・設備の運用変更、改善策の実施決定

- [ ] 目的、参加者、想定、行動・判断時刻及び中止・逸脱を記録した
- [ ] 設備・区域・人員を通常状態へ戻し、残留危険がない
- [ ] 評価結果、良好点、課題、原因及び改善案を記録した
- [ ] 改善担当、期限、承認者及び効果確認方法が受領された

訓練を実施しただけでは完了しない。評価と改善責任が確定し、通常運用への復帰が確認された状態を完了とする。
