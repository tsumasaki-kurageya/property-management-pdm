> 本文書は標準手順である。個別物件の警備計画、巡回経路、制限区域、鍵・撮影・個人情報の規程、緊急連絡及び警察・消防等への通報基準を優先する。

## 1. 目的

指定された時刻・経路・巡回点において、施錠、火気、照明、防犯・防災設備の外観、人物・物品及び施設状態を確認する。異常を早期に発見し、警備員自身と利用者の安全を優先して、接近回避、区域管理、応援・緊急連絡及び証跡保全へつなぐ。未確認箇所と経路変更を含め、勤務間で追跡可能にする。

## 2. 業務カタログとの関係

| 区分 | 業務ID | 業務名 | 本手順との関係 |
|---|---|---|---|
| 主業務 | `BM-11-04` | 巡回警備を行う | 指定経路・巡回点を確認する |
| 関連業務 | `BM-11-07` | 防災設備を確認する | 外観・表示等の承認済み確認を行う |
| 関連業務 | `BM-11-10` | 警備・防災記録を報告する | 巡回結果と異常を報告する |
| 関連業務 | `BM-13-01`〜`BM-13-05` | 作業記録・確認 | 巡回時刻、事実、写真、完了を残す |
| 関連業務 | `BM-17-06`〜`BM-17-07` | 危険予知、事故・ヒヤリハット管理 | 巡回前の安全確認と事象管理を行う |

## 3. 適用範囲

### 3.1 適用対象

- 建物内外の公開区域、共用部、外周及び許可された制限区域の定時・臨時巡回
- 扉・窓・シャッター等の施錠、火気・異臭・煙、照明、漏水、損傷、不審物等の目視確認
- 巡回点端末、タグ、記録票等による通過・確認記録
- 異常発見時の安全な範囲での初動、区域管理、応援要請及び引渡し

### 3.2 適用除外・使用禁止条件

- 契約・教育で認められていない身体拘束、所持品確認、危険人物の追跡・制圧
- 危険物・不審物の移動、開封又は接触
- 火災、ガス漏れ、倒壊、暴力、凶器等の生命危険がある区域への単独接近
- 設備の分解、専門操作又は復旧判断
- 許可されていない区域への入室、不要な映像・個人情報の取得

## 4. 実施体制

| 役割 | 主な責任 | 必要資格・教育 | 単独実施可否 |
|---|---|---|---|
| 巡回警備員 | 巡回、確認、記録、安全確保、応援要請 | 警備員教育、物件教育、緊急時・通信訓練 | 警備計画による |
| 警備責任者 | 指示、経路変更承認、初動統制、報告確認 | 責任者要件、物件計画の理解 | 該当なし |
| 施設責任者 | 区域閉鎖、利用制限、運用継続等の決定 | 物件別権限 | 該当なし |
| 設備・防災担当 | 設備異常、防災設備、火気等の専門判断 | 対象に必要な資格・力量 | 該当なし |
| 警察・消防・救急 | 公的対応と現場指揮 | 公的機関 | 該当なし |

## 5. 開始条件

- [ ] 有効な警備計画、巡回指示、予定時刻、経路及び巡回点を確認した
- [ ] 継続案件、工事、残留者、要注意区画、設備停止、臨時指示を引き継いだ
- [ ] 鍵、入館権限、通信機器、ライト、記録端末及び指定装備を照合した
- [ ] 単独勤務条件、定時連絡、応援要請方法及び緊急連絡を確認した
- [ ] 通行止め、避難経路、天候、暗所その他の危険を確認した
- [ ] `PROC-COM-002`による作業前安全確認を完了した

## 6. 必要資料・装備

| 区分 | 内容 | 確認事項 |
|---|---|---|
| 資料 | 警備計画、巡回経路図、巡回点一覧、残留者・工事情報、連絡網 | 最新版、当日変更、機密管理 |
| 鍵・認証 | 鍵、カード、巡回タグ等 | 数量、権限、有効性、持出記録 |
| 通信・照明 | 無線、電話、ライト、予備電源 | 動作、充電、通信可能範囲 |
| 安全装備 | 反射材、防護具、天候対策等 | 物件・時間帯・区域への適合 |
| 記録 | 巡回端末、記録票、時計、必要時カメラ | 時刻同期、保存先、撮影制限 |

## 7. 作業前確認・危険予知

| 危険・影響要因 | 予防措置 | 発生時の初動 |
|---|---|---|
| 不審者・暴力・凶器 | 距離・退路を確保し、単独接近を避ける | 退避、応援・警察連絡、利用者誘導 |
| 不審物・危険物 | 触れず、無線等の使用制限を含む物件基準を確認 | 区域確保、退避、責任者・公的機関へ |
| 火災・煙・ガス・倒壊 | 避難経路と通報方法を確認する | 接近せず通報、避難支援、消防計画へ |
| 暗所・段差・屋外 | ライト、足元、天候、閉鎖箇所を確認する | 経路変更、区域規制、負傷時救護 |
| 単独勤務・通信不能 | 定時連絡、位置共有、応援到着目標を定める | 安全地点へ戻り巡回中止・上申 |
| 利用者・個人情報 | 必要最小限の声掛け・記録・撮影とする | 不要な取得を中止し責任者へ確認 |

## 8. 標準手順

| No. | 実施内容 | 確認・判定 | 異常・未達時 | 記録・証跡 |
|---:|---|---|---|---|
| 1 | 勤務記録、監視情報、引継ぎから巡回前状態を確認する | 継続案件・臨時指示を理解している | 不明点は開始保留 | 開始時刻、指示ID |
| 2 | 巡回開始を記録し、指定経路へ入る | 通信・装備・退路が確保されている | 条件未達は中止・上申 | 開始、装備照合 |
| 3 | 各巡回点で扉、窓、シャッター、鍵・封印等を確認する | 所定状態で損傷・こじ開け痕がない | むやみに触れず9・10章へ | 巡回点、時刻、状態 |
| 4 | 火気、煙、異臭、漏水、照明、通路、設備外観を確認する | 危険・通行障害・異常表示がない | 人命危険時は退避・緊急連絡 | 観察事実、位置 |
| 5 | 人物、車両、物品を客観的に観察する | 許可情報・通常状態と矛盾がない | 安全距離を保ち応援要請 | 外見、行動、時刻。推定と区別 |
| 6 | 巡回点の通過・確認を記録する | 時刻・場所・実施者が追跡できる | 端末不良時は代替記録 | タグ、端末、記録票 |
| 7 | 経路変更・未確認箇所を記録する | 理由、影響、再確認予定が明確 | 無断で危険区域へ入らない | 変更理由、承認、予定 |
| 8 | 発見事象に対し安全な範囲で区域管理・連絡を行う | 自身と利用者の安全を損なわない | 接近・追跡せず引渡す | 連絡時刻、措置、回答 |
| 9 | 巡回終了時に鍵・装備を照合し、結果を報告する | 紛失・未返却がなく、未解決事項を引き継いだ | 不一致は即時報告 | 終了、照合、引継ぎ |

## 9. 判定基準

| 状態 | 手順固有の基準 | 標準対応 | 完了可否 |
|---|---|---|---|
| 正常 | 指定点を確認し、施錠・火気・人物・物品・施設状態が基準どおり | 記録して継続 | 可 |
| 要観察 | 軽微な不一致、反復事象、死角等があるが直ちに危険でない | 重点監視と期限付き引継ぎ | 条件付き可 |
| 異常 | 施錠不良、無断立入、不審物、設備異常、事故等がある | 安全距離、応援・責任者連絡、区域管理 | 原則不可 |
| 危険 | 暴力、凶器、火災、ガス、重大事故、倒壊等の生命危険 | 接近回避、退避、緊急通報、避難支援 | 不可 |

「異常なし」は、指定巡回点を確認できた場合に限る。未確認点がある場合は、その理由と再確認計画を別に記録する。

## 10. 異常時対応

| 事象 | 安全確保・一次対応 | 中止・隔離条件 | 報告・後続 |
|---|---|---|---|
| 施錠不良・侵入痕 | むやみに触れず、周囲と退路を確認する | 侵入者の可能性、破壊、危険物 | 警備責任者・施設責任者、必要時警察へ |
| 不審者 | 距離を保ち、退路・応援を確保する | 暴力、凶器、威嚇、複数人 | 単独対応を避け、応援・警察連絡 |
| 不審物 | 触れず、周囲を遠ざける | 内容不明、異臭、配線、脅迫情報等 | 区域管理、責任者・警察等へ |
| 火災・煙・ガス臭 | 安全地点から通報し避難を支援する | 常に通常巡回を中止 | 消防計画、消防・施設責任者へ |
| 負傷者・急病人 | 自身の安全を確認し、訓練範囲で救護・通報 | 二次災害、暴力、危険環境 | 救急・責任者へ、現場保全 |
| 漏水・設備異常 | 立入防止し設備担当へ連絡する | 感電、崩落、大量漏水等 | 設備異常対応へ引渡し |
| 巡回不能・通信断 | 安全地点へ戻り代替連絡を試みる | 単独勤務条件を満たさない | 巡回中止、再計画・引継ぎ |

## 11. 権限・上申

| 判断・行為 | 権限 | 標準的な扱い |
|---|---|---|
| 承認済み経路の巡回・照合・記録 | L0 | 巡回警備員が実施 |
| 安全な再確認、声掛け、立入保留、応援要請、初期区域管理 | L1 | 教育・契約・安全の範囲内で実施 |
| 経路変更、長時間の区域閉鎖、施設運用変更 | L2 | 警備・施設責任者の承認 |
| 契約外対応・追加費用 | L3 | 契約権限者が決定 |
| 重大リスク受容・恒常的な警備計画変更 | L4 | オーナー等の権限者が決定 |
| 退避・緊急通報 | 緊急権限 | 切迫時は承認を待たず実施し即時報告 |
| 区域の利用再開・現場解除 | L2又は公的機関 | 警察・消防等へ引渡した場合はその指示を優先 |

## 12. 記録・証跡・完了条件

- [ ] 巡回開始・終了、実施者、予定と実績、各巡回点を記録した
- [ ] 経路変更、未確認箇所、理由、承認及び再確認予定を記録した
- [ ] 発見事項は時刻、場所、人物・物品・状態、発言等の観察事実として記録した
- [ ] 推定と事実を区別し、訂正履歴を残した
- [ ] 写真・映像は必要性、撮影権限及び保存先を確認した
- [ ] 応援、施設責任者、警察・消防・救急等への連絡時刻、内容、回答を記録した
- [ ] 証跡、遺失物、現場等の引渡し先と時刻を記録した
- [ ] 鍵・装備を照合し、未解決事項を次勤務者へ引き継いだ

警報復帰、対象者の退出又は公的機関の到着だけで案件を終了せず、指揮の引渡し、現場保全、残課題及び次勤務者への引継ぎを確認する。

## 13. 適用差分

| 分析軸 | 主な差分 |
|---|---|
| 建物用途 | 商業施設の群集・営業時間、病院の患者・重要区画、ホテルの24時間利用、オフィスの残業者等を反映する |
| 管理方式 | 巡回・機械警備では駆付け条件、到着目標、遠隔情報、単独勤務、応援到着時間を追加する |
| 契約役割 | 警備会社、元請BM、施設責任者、設備担当間の指揮・報告・証跡引渡しを明示する |
| 法令義務 | 警備員教育、警備計画、消防・防災、個人情報・映像、記録保存等を確認する |
| 責任分界 | 入館拒否、区域閉鎖、避難、営業継続、情報開示、公的機関対応の決定先を定める |
