# ビルメンテナンス法令義務プロファイル v1.0

## 1. 目的

本書は、[ビルメンテナンス業務カタログ](./building-maintenance-business-catalog.md)に整理した共通業務について、建物・設備・用途・所在地等に応じて発生する法令上の義務を分析するための補助カタログである。

法定業務を単なる「法定点検」の一覧として扱わず、対象判定、選任、届出、維持管理、点検・測定・検査、行政報告、記録保存、不適合是正までを一つのライフサイクルとして整理する。

本書は特定の製品・システムの機能やカバレッジを評価するものではない。また、個別物件に対する法的判断を示すものではない。実際の適用に当たっては、最新の法令、告示、自治体条例、所管行政庁の運用及び専門家の判断を確認する。

## 2. 対象範囲と前提

- 対象は、ビルメンテナンス会社が受託業務として関与し得る建築物・設備の維持管理上の法令義務である。
- 法令上の義務主体が、ビルメンテナンス会社であるとは限らない。所有者、管理者、占有者、設置者、事業者、管理権原者等に残る義務がある。
- 契約により作業を委託・再委託しても、法令上の義務が当然に移転するとは限らない。
- 法令名が同じでも、用途、面積、階数、収容人員、設備能力、設置形態、所在地等により対象・周期・様式が異なる。
- 国の法令だけでなく、条例、特定行政庁、消防本部、保健所等の指定・運用を確認する。
- 法令改正に備え、義務の内容だけでなく、根拠条項、適用条件、有効期間及び確認日を記録する。

## 3. 用語定義

| 用語 | 定義 |
|---|---|
| 法令義務 | 法律、政令、省令、告示、条例等により、一定の主体へ課される行為又は状態維持の義務 |
| 法定業務 | 法令義務を履行するために行う選任、届出、維持管理、点検、測定、検査、報告、保存、是正等の業務 |
| 法定点検 | 法定業務のうち、法令に基づいて建物・設備・管理状況を点検、調査又は検査する業務 |
| 法的義務主体 | 法令上、義務の履行を求められる所有者、管理者、占有者、設置者、事業者等 |
| 管理権原者 | 建物又はその部分の管理行為を法律、契約又は慣習上行う権原を有する者 |
| 選任者 | 法令上必要な管理者、主任技術者等を選任する者 |
| 実施者 | 点検、測定、清掃、訓練等を実際に行う者 |
| 資格者 | 法令上必要な資格、登録、免状又は知見を有する者 |
| 報告者 | 行政機関等へ正式な報告又は届出を行う者・名義人 |
| 是正責任者 | 不適合を解消し、適法又は基準適合状態へ戻す責任を持つ者 |
| 所管行政庁 | 特定行政庁、消防長・消防署長、保健所、産業保安監督部、労働基準監督署等の所管機関 |

## 4. 法令義務と契約業務の違い

契約上「点検一式」を受託していても、法令上の義務を誰が負うか、誰の名義で報告するか、誰が是正費用を負担するかは別に確認する。

| 区分 | 決定根拠 | 代表的な確認事項 |
|---|---|---|
| 法的義務 | 法令、告示、条例、行政運用 | 義務主体、資格、周期、報告先、保存期間 |
| 契約責任 | 元契約、再委託契約、仕様書 | 実施範囲、成果物、検収、費用、損害責任 |
| 作業分担 | 年間計画、作業計画、発注書 | 実施者、責任者、日程、手順、連絡経路 |
| 費用負担 | 契約、見積、承認、修繕区分 | 点検費、報告費、是正費、緊急対応費 |

一つの義務に対し、次の主体が別々になる場合がある。

| 主体属性 | 例 |
|---|---|
| 法的義務主体 | 建物所有者又は管理者 |
| 契約窓口 | 元請けビルメンテナンス会社 |
| 点検実施者 | 専門点検会社 |
| 点検資格者 | 専門点検会社に所属する資格者 |
| 結果確認者 | 元請けの業務責任者 |
| 行政報告名義 | 所有者又は管理者 |
| 是正実施者 | 修繕会社 |
| 是正費用負担者 | 契約で定める所有者等 |

詳細な契約上の役割分担は、[ビルメンテナンス契約役割プロファイル](./contract-role-profiles.md)と組み合わせて扱う。

## 5. 法令対応ライフサイクル

### 5.1 適用判定

1. 建物、区画、設備及び事業の属性を収集する。
2. 国法、政省令、告示、条例及び行政指定を候補として抽出する。
3. 用途、規模、能力、収容人員、所在地等の条件へ照合する。
4. 適用、非適用、要確認を判定する。
5. 判定根拠、確認資料、確認者及び確認日を記録する。

### 5.2 体制整備

1. 法的義務主体と管理権原者を特定する。
2. 必要な管理者、主任技術者、資格者及び登録事業者を特定する。
3. 選任、解任、兼任、外部委託等の方法を決定する。
4. 行政への選任・変更・廃止等の届出を行う。
5. 契約上の実施者、確認者、報告者及び是正責任者を定める。

### 5.3 計画・実施

1. 法定周期、起算日、指定月、猶予及び例外条件を確認する。
2. 年間計画へ点検、測定、清掃、訓練、報告等を展開する。
3. 資格者、日程、作業停止、テナント調整等を手配する。
4. 所定の方法、項目、測定器及び様式で実施する。
5. 実施結果、測定値、写真、資格者、使用機器等を記録する。

### 5.4 判定・報告・保存

1. 法定基準、判定基準及び行政運用に基づき結果を判定する。
2. 要是正、不適合、未実施、報告遅延等を登録する。
3. 所定の名義、期限、様式及び提出先で報告・届出する。
4. 受付、受理、差戻し、補正及び完了の状態を管理する。
5. 法定期間及び契約期間に従って証跡を保存する。

### 5.5 是正・再判定

1. 危険性と法令上の扱いから優先度を判定する。
2. 応急処置、使用停止、修繕、再検査等を計画する。
3. 責任者、期限、費用負担及び承認者を決定する。
4. 是正後の適合確認と行政への追加報告を行う。
5. 用途・設備・所有者・管理会社・法令の変更時に適用を再判定する。

## 6. 法令義務マスター

法令義務は「法令名＋周期」の一行ではなく、次の属性を持つマスターとして管理する。

| 属性 | 内容 |
|---|---|
| 義務ID | 義務を一意に識別する内部ID |
| 義務名称 | 人が識別しやすい名称 |
| 法令領域 | 建築、防火、衛生、電気、環境、労働安全等 |
| 根拠法令 | 法律、政令、省令、告示、条例、条項 |
| 根拠資料URL | 行政機関の法令・制度・様式ページ |
| 有効期間 | 適用開始日、終了日、改正施行日 |
| 適用条件 | 用途、面積、階数、収容人員、設備能力、設置形態等 |
| 地域条件 | 都道府県、市区町村、特定行政庁、消防本部等 |
| 対象単位 | 建物、区画、設備、系統、事業場、管理権原者等 |
| 法的義務主体 | 所有者、管理者、占有者、設置者、事業者等 |
| 義務種別 | 選任、届出、維持管理、点検、測定、検査、訓練、報告、保存、是正等 |
| 実施資格 | 資格者、登録事業者、知見を有する者等 |
| 実施条件 | 方法、項目、測定位置、使用機器、立会い等 |
| 周期・期限 | 定期周期、イベント起点の期限、行政指定月等 |
| 起算基準 | 前回実施日、年度、設置日、使用開始日、指定日等 |
| 報告名義・報告先 | 提出主体と所管行政庁 |
| 様式・添付資料 | 国様式、自治体様式、図面、写真、資格証等 |
| 保存期間 | 記録・報告書・整備履歴等の保存要件 |
| 不適合時措置 | 是正、再検査、使用停止、速報等 |
| 委託条件 | 委託可否、承認、登録、資格、再委託制限等 |
| 変更トリガー | 用途、面積、設備、能力、所有者、法令等の変更 |
| 確認状態 | 適用、非適用、要確認、失効等 |
| 確認者・確認日 | 判定責任者と最終確認日 |

周期は表示用文字列だけでなく、起算基準、期限計算方法、例外条件及び地域差を構造化する。法令改正後も過去実績を説明できるよう、マスターは有効期間付きで版管理する。

## 7. 物件別の適用判定

### 7.1 判定に必要な物件属性

| 分類 | 主な属性 |
|---|---|
| 所在地 | 都道府県、市区町村、特定行政庁、消防本部、保健所 |
| 建物 | 用途、複合用途、延べ面積、用途別面積、階数、地下階、構造、竣工日 |
| 利用 | 収容人員、就寝利用、不特定多数利用、営業時間、学校種別等 |
| 防火 | 防火対象物区分、管理権原、消防用設備、防火設備、避難施設 |
| 衛生 | 特定用途、特定用途面積、空調方式、給水方式、受水槽有効容量 |
| 電気 | 受電電圧、最大電力、発電・蓄電設備、事業場区分 |
| 冷凍空調 | 第一種特定製品、圧縮機定格出力、冷媒、設置・廃棄状態 |
| 排水 | 浄化槽の有無、処理方式、処理対象人員、使用開始日 |
| 労働安全 | ボイラー、圧力容器、クレーン、昇降設備等の種類・能力 |
| 事業 | 所有者、占有者、設置者、事業者、管理権原者、契約関係 |

### 7.2 判定結果

| 判定 | 意味 | 次の行動 |
|---|---|---|
| 適用 | 条件を満たし義務が発生する | 担当主体、期限、実施計画を登録する |
| 非適用 | 条件を満たさない | 根拠と判定日を保存する |
| 要確認 | 情報不足又は行政判断が必要 | 不足情報と問い合わせ先を登録する |
| 条件付き | 一部区画、設備又は期間のみ適用 | 対象範囲と条件を明示する |
| 失効 | 撤去、用途変更、法改正等により終了 | 終了日と証跡を保存する |

「非適用」も根拠付きで保存する。対象外であることを再調査せず説明でき、用途・設備・法令の変更時に影響対象を逆引きできるためである。

## 8. 代表的な法令義務プロファイル

以下は、ビルメンテナンス業務との接続が強い代表領域を共通形式で整理したものである。適用条件・周期・様式は個別条件や自治体運用で変わるため、実物件では必ず所管行政庁の情報へ照合する。

### 8.1 建築基準法：定期報告制度

| 項目 | 内容 |
|---|---|
| 主な対象 | 特定建築物、建築設備、防火設備、昇降機等 |
| 主な義務主体 | 所有者又は管理者 |
| 義務 | 資格者による調査・検査、結果の特定行政庁への報告、指摘事項の是正 |
| 資格 | 一級・二級建築士又は対象区分に応じた法定資格者 |
| 地域差 | 国が一律指定する対象に加え、特定行政庁が対象、時期、様式等を定める場合がある |
| 主な証跡 | 調査・検査結果表、概要書、図面、写真、報告受付記録、是正記録 |

管理上は、少なくとも次を別義務として登録する。

- 特定建築物の定期調査
- 建築設備の定期検査
- 防火設備の定期検査
- 昇降機等の定期検査
- 行政指摘事項の是正及び完了報告

国土交通省は、国指定対象のほか地域の実情に応じて特定行政庁が対象を定め得ること、調査・検査には法令に基づく資格が必要であることを示している。2025年7月1日施行の調査・検査項目等の改正も掲載されているため、古い点検表の継続利用を避ける必要がある。([国土交通省「建築基準法に基づく定期報告制度」](https://www.mlit.go.jp/jutakukentiku/build/jutakukentiku_house_tk_000039.html))

### 8.2 消防法：防火管理・消防用設備等

| 項目 | 内容 |
|---|---|
| 主な対象 | 防火対象物、消防用設備等、防災管理対象物等 |
| 主な義務主体 | 所有者、管理権原者、防火対象物の関係者等（義務ごとに異なる） |
| 義務 | 管理者選任、消防計画、訓練、設備点検、防火・防災管理点検、報告、維持管理 |
| 資格 | 防火管理者、防災管理者、消防設備士、消防設備点検資格者、防火対象物点検資格者等 |
| 報告先 | 消防長又は消防署長等 |
| 主な証跡 | 選任届、消防計画、訓練記録、点検結果報告書、改修報告、維持台帳 |

少なくとも次を別義務として扱う。

- 防火管理者・防災管理者・統括防火管理者等の選任及び届出
- 消防計画の作成・変更及び届出
- 消火・通報・避難訓練等の計画、実施及び事前通報
- 消防用設備等の機器点検及び総合点検
- 消防用設備等点検結果の行政報告
- 防火対象物点検及び防災管理点検
- 消防用設備等の不良・不適合の改修

消防用設備等は一般に機器点検を6か月ごと、総合点検を1年ごとに行う。行政への結果報告周期は防火対象物の区分等により異なるため、点検周期と報告周期を別フィールドで管理する。([総務省消防庁「消防用設備等点検報告制度とは」](https://www.fdma.go.jp/mission/prevention/items/prevention001_19_hutekisetsu_leaflet.pdf))

### 8.3 建築物衛生法：特定建築物の環境衛生管理

| 項目 | 内容 |
|---|---|
| 主な対象 | 政令で定める用途・規模の特定建築物 |
| 主な義務主体 | 所有者、占有者その他、維持管理について権原を有する者 |
| 義務 | 届出、管理技術者選任、環境衛生管理基準に従った維持管理、帳簿・図面等の整備 |
| 資格・体制 | 建築物環境衛生管理技術者、必要に応じ登録事業者等 |
| 主な領域 | 空気環境、給水・排水、清掃、ねずみ・昆虫等の防除 |
| 主な証跡 | 特定建築物届、選任記録、測定・検査結果、清掃記録、設備図面、立入検査対応記録 |

代表的な業務は次のとおりである。

- 特定建築物該当判定及び使用開始・変更等の届出
- 建築物環境衛生管理技術者の選任
- 空気環境測定及び基準逸脱時の調整・再測定
- 冷却塔、加湿装置、排水受け等の点検・清掃
- 飲料水の残留塩素測定、水質検査、貯水槽清掃
- 排水設備の清掃
- 日常清掃・大掃除
- ねずみ・昆虫等の生息調査及び防除
- 帳簿、図面、測定・作業記録の整備
- 保健所等の報告徴収・立入検査対応

厚生労働省は、空気環境測定を2か月以内ごとに1回、冷却塔・加湿装置等の一定の点検を使用期間中1か月以内ごとに1回、飲料水の残留塩素検査を7日以内ごとに1回とするなど、項目ごとに異なる周期を示している。これらを「衛生点検：年1回」のように集約してはならない。([厚生労働省「建築物環境衛生管理基準」](https://www.mhlw.go.jp/bunya/kenkou/seikatsu-eisei10/))

### 8.4 水道法：専用水道・簡易専用水道等

| 項目 | 内容 |
|---|---|
| 主な対象 | 専用水道、簡易専用水道等 |
| 主な義務主体 | 設置者等 |
| 義務 | 管理、水質管理、貯水槽清掃、定期検査、異常時措置等 |
| 実施者 | 設置者、登録検査機関、清掃事業者等 |
| 主な証跡 | 施設台帳、清掃記録、水質結果、検査結果、異常対応記録、行政連絡記録 |

管理上は、建築物衛生法による給水管理と、水道法による簡易専用水道等の義務を別に判定し、重複する作業は一つの実施記録から複数義務へ関連付ける。

簡易専用水道の管理検査は1年以内ごとに1回とされる。受水槽の容量、給水方式、地域条例等を確認し、対象判定と清掃・検査を別の義務として管理する。([厚生労働省「水道法の一部改正に伴う簡易専用水道の規制等について」](https://www.mhlw.go.jp/web/t_doc?dataId=00ta5425&dataType=1&pageNo=1))

### 8.5 電気事業法：自家用電気工作物の保安

| 項目 | 内容 |
|---|---|
| 主な対象 | 自家用電気工作物を含む事業用電気工作物等 |
| 主な義務主体 | 電気工作物の設置者 |
| 義務 | 技術基準適合維持、保安規程、主任技術者、巡視・点検・測定、事故対応、届出・記録 |
| 体制 | 主任技術者の選任、選任許可・兼任承認又は保安管理業務の外部委託承認等 |
| 報告先 | 経済産業省・産業保安監督部等 |
| 主な証跡 | 保安規程、選解任届等、点検記録、事故報告、工事・変更記録、立入検査資料 |

電気設備の点検周期を一律の法定値として登録せず、承認・届出された保安規程、設備区分、外部委託承認制度の条件等を根拠として物件別に管理する。

経済産業省は、事業用電気工作物について技術基準適合維持、保安規程の作成・届出・遵守、主任技術者選任等の義務を示している。([経済産業省「電気工作物の保安」](https://www.meti.go.jp/policy/safety_security/industrial_safety/sangyo/electric/detail/setsubi_hoan.html))

### 8.6 フロン排出抑制法：業務用冷凍空調機器

| 項目 | 内容 |
|---|---|
| 主な対象 | 第一種特定製品である業務用冷凍空調機器 |
| 主な義務主体 | 機器の管理者 |
| 義務 | 適切な設置・使用、簡易点検、一定規模以上の機器の定期点検、漏えい時の修理、充塡・回収、記録、算定漏えい量報告、廃棄時管理 |
| 資格・体制 | 定期点検は十分な知見を有する者、充塡・回収は登録業者等 |
| 主な証跡 | 機器別点検記録簿、修理・充塡・回収履歴、漏えい量算定資料、行程管理票等 |

対象機器は、一般の設備台帳とは別に、第一種特定製品該当性、圧縮機定格出力、冷媒、管理者、設置・移設・譲渡・廃棄状態を保持する。

環境省は、全管理者に簡易定期点検、一定規模以上の機器に専門家による定期点検を求め、漏えい確認時は原則として充塡前の修理を求めている。また、機器ごとの点検・整備記録は、廃棄等を行い冷媒の引渡しを完了した日から3年を経過するまで保存する。([環境省「機器の点検」](https://www.env.go.jp/earth/furon/operator/isshu_2.html)、[環境省「点検・整備の記録」](https://www.env.go.jp/earth/furon/operator/isshu_3.html))

## 9. 自治体条例・個別指定の扱い

次の情報は国法マスターへ上書きせず、地域ルールとして別階層で管理する。

- 特定行政庁が指定する定期報告対象、周期、報告月及び様式
- 消防本部・消防署が定める届出方法、様式、運用上の確認事項
- 保健所・自治体が定める特定建築物、簡易専用水道、浄化槽等の手続
- 廃棄物、騒音、排水、景観、屋外広告物等の条例
- 行政庁から個別物件へ出された指示、改善命令、特例認定等

地域ルールには、適用地域、上位義務ID、所管機関、根拠URL、発効日、失効日及び確認日を持たせる。同じ市区町村でも所管が複数に分かれる場合があるため、「自治体名」だけでなく所管行政庁まで特定する。

## 10. 点検・報告・是正・保存の管理

### 10.1 点検と行政報告を分離する

| 状態 | 例 |
|---|---|
| 実施予定 | 点検日は確定したが未実施 |
| 実施済み | 現地作業は完了したが結果未確定 |
| 判定済み | 資格者が結果を確定 |
| 社内確認済み | 元請け・管理者が成果物を確認 |
| 提出済み | 行政へ提出したが未受理 |
| 受理済み | 受付・受理を確認 |
| 補正中 | 行政からの差戻し・補正依頼へ対応中 |
| 是正中 | 不適合の改修・再検査を実施中 |
| 完了 | 報告・是正・証跡保存まで完了 |

点検を実施しても、報告、受理、是正が完了していなければ義務履行が完了していない場合がある。

### 10.2 未実施・期限超過

未実施又は期限超過を検出した場合は、次を記録する。

- 対象義務と対象物
- 法定期限、契約期限及び超過日数
- 未実施理由
- 危険性、営業影響及び法令上の影響
- 法的義務主体、契約窓口及びエスカレーション先
- 応急措置、再計画日及び行政相談の要否
- 顧客報告及び承認記録

### 10.3 是正管理

点検指摘は、単なる修繕依頼ではなく次の関連を保持する。

`法令義務 → 点検実績 → 指摘事項 → 是正計画 → 修繕作業 → 再確認 → 行政報告`

是正状態は、未対応、応急措置済み、見積中、承認待ち、施工予定、施工済み、再確認済み、行政確認済み等に分ける。所有者の承認待ち等により停滞する場合も、法的義務主体と期限を失わないよう管理する。

### 10.4 証跡保存

保存対象は少なくとも次の単位で関連付ける。

- 法令義務と適用判定
- 対象建物、区画、設備、系統
- 実施日、実施者、資格者
- 使用した点検表・様式の版
- 測定器、校正情報、測定値
- 写真、図面、報告書、届出書
- 行政の受付・受理・補正記録
- 指摘事項、是正及び再確認
- 保存期限と廃棄予定日

契約終了や管理会社変更時にも法定記録が失われないよう、引継ぎ対象、原本保管者、写しの提供範囲及び廃棄権限を定める。

## 13. 更新・レビュー方針

- 法令義務マスターは少なくとも定期的に見直し、改正公布時と施行前にも影響確認する。
- 物件別適用台帳は、用途、面積、設備、能力、所有者、管理者、契約、行政指定等の変更時に再判定する。
- 根拠URLだけに依存せず、条項、告示番号、様式版、確認日を記録する。
- 行政機関への確認事項は、問い合わせ日、回答部署、回答内容、回答者及び関連資料を保存する。
- 法令解釈をビルメンテナンス会社だけで確定できない場合は、義務主体、所管行政庁又は有資格専門家へエスカレーションする。
- 本書自体も法令改正に合わせて改訂し、改訂時点と主な変更点を履歴化する。
