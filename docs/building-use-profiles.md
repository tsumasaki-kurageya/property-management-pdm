# 建物用途別ビルメンテナンス業務プロファイル v1.0

## 1. 目的

本書は、[ビルメンテナンス業務カタログ](./building-maintenance-business-catalog.md)に整理した共通業務が、建物用途によってどのように変化するかを分析するための補助カタログである。

用途ごとに業務を複製するのではなく、共通業務IDを維持したまま、適用条件、実施頻度、時間制約、品質基準、体制、調整先、証跡などの差分を記述する。

本書は特定の製品・システムの機能やカバレッジを評価するものではない。

## 2. 対象範囲と前提

- 対象は、ビルメンテナンス会社が受託し得る建物管理業務である。
- 用途分類は業務分析用であり、建築基準法、消防法その他の法令上の用途区分を代替しない。
- 実際の業務は、用途だけでなく、規模、設備、運用時間、利用者、管理方式、契約範囲、法令・条例等で決まる。
- 法的義務主体、資格、周期、届出・報告先は、個別物件について最新の法令と所管行政庁の定めを確認する。
- 病院内の医療行為、工場の生産設備保全、ホテルの客室運営など、施設事業者の本業は対象外とし、建物管理と接続する部分のみ扱う。

## 3. 分析モデル

### 3.1 二層構造

業務分析は次の二層で行う。

1. **共通業務層**：業務カタログの181業務。業務の目的と基本的な完了条件を定義する。
2. **用途プロファイル層**：用途ごとに、共通業務の実施条件や要求水準の差分を定義する。

用途差だけでは説明できない条件は、後述する物件条件で補正する。

### 3.2 用途差を記述する属性

| 属性 | 確認内容 |
| --- | --- |
| 適用区分 | 標準的に実施するか、設備・規模・契約等の条件付きか、通常は対象外か |
| 実施頻度 | 常時、都度、日次、週次、月次、季節、休館時など |
| 実施可能時間 | 営業・執務・診療・授業・居住への影響を避ける時間帯 |
| 初動・復旧要求 | 異常受付、現場到着、応急処置、復旧に求められる時間 |
| 品質基準 | 美観、衛生、温湿度、騒音、臭気、安全、可用性等の要求水準 |
| 手順・禁止事項 | ゾーニング、養生、薬剤、工具、服装、入室、撮影等の制約 |
| 管理体制 | 常駐・巡回、24時間対応、複数名作業、専門要員等 |
| 調整・承認先 | オーナー、テナント、店舗、医療部門、学校、居住者等 |
| 証跡 | 写真、測定値、作業票、鍵授受、入退室、承認、引継ぎ等 |
| 差分要因 | 利用者、運用時間、設備、事業継続、安全、法令、契約等 |

「適用区分」「頻度」「重要度」は分けて判断する。多くの用途に共通する業務でも、用途によって頻度や失敗時の影響は大きく異なる。

## 7. 用途別プロファイル

### USE-01 オフィスビル

**業務差の中心**：複数テナントの執務を妨げず、共用設備とテナント専有部の責任分界を保つこと。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 作業時間 | 始業前、夜間、休日を基本枠とし、騒音・臭気・停電作業はテナント別に調整する | BM-04-04、BM-04-07〜08 |
| 環境 | 在館率、外気、テナント要望を考慮して空調・照明を運転する | BM-08-01〜02、BM-08-08 |
| 問い合わせ | 温冷感、照明、カード・鍵、清掃等をテナント責任者経由または利用者から受け付ける | BM-12-01〜07 |
| 証跡 | 作業届、入退館、鍵授受、テナント承認、停電対象と復旧確認を残す | BM-11-02〜03、BM-13 |

### USE-02 商業施設

**業務差の中心**：多数の来館者の安全と店舗営業を維持しながら、共用部、飲食区画、バックヤードを異なる頻度・手順で管理すること。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 清掃・廃棄物 | トイレ、フードコート、出入口等を営業中も巡回し、厨房由来廃棄物・臭気・害虫を管理する | BM-06-02〜10、BM-07-09〜10 |
| 設備 | 営業時間、催事、季節、店舗負荷に合わせ、空調・給排水・搬送設備を運転する | BM-08、BM-09 |
| 異常対応 | 漏水、床面危険、エスカレーター、停電等では来館者の隔離・誘導と営業影響の抑制を同時に行う | BM-10-02〜03、BM-11-05〜06 |
| 調整 | 店舗、運営会社、警備、清掃、設備、催事担当の連絡系統を明確にする | BM-03-04〜05、BM-12 |

### USE-03 ホテル・宿泊施設

**業務差の中心**：24時間の宿泊機能を保ち、客室販売・宴会・飲食の稼働に応じて静穏かつ迅速に対応すること。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 作業時間 | 客室在室・チェックイン／アウト、宴会予定を踏まえて入室・騒音作業を調整する | BM-04、BM-12-08 |
| 設備・衛生 | 給湯、空調、給排水、厨房、浴場等の連続需要と衛生を管理する | BM-07、BM-08、BM-09 |
| 異常対応 | 客室交換や代替提供を担う運営部門と連携し、宿泊客への影響を最小化する | BM-10-01〜03、BM-12-03〜07 |
| セキュリティ | 宿泊区域、従業員区域、宴会・搬入口を分け、客室入室と鍵の証跡を厳格化する | BM-11-02〜04、BM-13-02 |

### USE-04 病院・医療施設

**業務差の中心**：診療継続と患者安全を優先し、感染対策部門・医療部門が定める区域別手順に従うこと。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| ゾーニング | 一般、診療、手術、隔離等の区域ごとに清掃手順、用具、保護具、動線、廃棄区分を定義する | BM-03-05〜06、BM-06、BM-17-05〜06 |
| 設備停止 | 電源、空調、給水、医療ガス等の影響先と代替手段を医療部門が確認してから作業する | BM-04-08、BM-10-02〜10 |
| 体制 | 24時間の警報・一次対応、重要設備のエスカレーション、感染対策教育を組み込む | BM-05-04〜06、BM-08-03〜07 |
| 証跡 | 区域、接触予防策、作業者、使用資材、測定値、医療部門承認を追跡できるようにする | BM-13、BM-17-09 |

医療施設では、施設管理者の責任の下で感染対策委員会・感染対策チーム等が組織的に対応し、部門固有の対策を含むマニュアルを整備する考え方が示されている。ビルメンテナンス業務は当該体制と手順に従って接続する。

### USE-05 学校・教育施設

**業務差の中心**：児童・生徒等の安全を最優先に、授業・試験・学校行事と作業を分離すること。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 計画 | 騒音、停電、薬剤、高所作業等を授業外または長期休業中に配置する | BM-04、BM-17-05〜06 |
| 安全 | 工具・薬剤・作業区画を確実に隔離し、登下校・休み時間の動線を考慮する | BM-11-04〜06、BM-17-07 |
| 設備・衛生 | 教室環境、トイレ、給食設備、体育館、プール等を設備の有無に応じて管理する | BM-06〜09 |
| 防災 | 教職員、児童生徒、地域利用者を考慮し、学校側の計画と訓練に接続する | BM-11-07〜10 |

### USE-06 共同住宅

**業務差の中心**：24時間の生活を尊重し、共用部分と専有部分、管理組合・管理会社・居住者の権限を区別すること。

**業務差の中心**：車両・荷役・保管業務を止めずに、広い床面、搬出入口、シャッター、防火区画等を安全に維持すること。

**業務差の中心**：建物付帯設備と生産・研究設備の責任境界を明確にし、危険物、機密、品質、操業停止の制約下で作業すること。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 設備分界 | 電力、蒸気、冷却水、排気等のユーティリティと生産設備の境界・停止権限を定義する | BM-02-02、BM-03-02〜05、BM-14-04 |
| 作業許可 | 火気、電気、薬品、酸欠、高所等の許可、ロックアウト等の施設ルールに従う | BM-04-08、BM-17-05〜07 |
| 変更管理 | 設定変更・部品交換・工事が品質や研究結果へ与える影響を利用部門と事前評価する | BM-09-07〜10、BM-10-07〜10 |
| 証跡 | 区域入退室、教育、持込品、測定器校正、作業前後状態、承認を残す | BM-05-06、BM-13、BM-15-07〜08 |

### USE-09 データセンター

**業務差の中心**：冗長構成を保ったまま電源・冷却を連続運用し、人的作業によるサービス停止を防ぐこと。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 監視・体制 | 電源、冷却、漏水、火災等を24時間監視し、冗長性低下も異常として扱う | BM-05-04、BM-08-03〜07 |
| 変更管理 | 対象系統、冗長構成、切戻し、禁止操作、立会者を作業計画に明記する | BM-04-08、BM-09-10、BM-10-07〜10 |
| 入退室 | 承認済み作業者だけを区域別に許可し、共連れ、持込・持出、作業時刻を記録する | BM-11-02〜04、BM-13-02 |
| 復旧 | サービス影響と冗長性低下を即時に連絡し、段階的なエスカレーションを行う | BM-10-02〜03、BM-12-06〜09 |

### USE-10 公共・文化施設

**業務差の中心**：催事ごとに変わる利用人数・時間・動線・環境条件へ対応し、不特定多数の安全を確保すること。

| 観点 | 用途固有の差分 | 関連業務 |
| --- | --- | --- |
| 催事連動 | 開館、休館、搬入出、リハーサル、公演・展示の予定を運転・清掃・警備計画へ反映する | BM-04、BM-08-01〜02、BM-11-01 |
| 群集安全 | 入退場、行列、避難経路、定員、要配慮者を施設運営側と管理する | BM-11-02、BM-11-05〜10 |
| 専門区域 | 舞台、収蔵、展示、音響・照明等は専門業者・学芸部門との責任分界を定める | BM-03-04〜05、BM-05-07〜09 |
| 環境・保全 | 展示物等に固有の温湿度・照度・振動条件がある場合、一般快適性と分けて管理する | BM-07、BM-08、BM-13-03 |

### USE-11 複合用途施設

**業務差の中心**：用途区画ごとの異なる要求と、建物全体で共有する設備・避難・搬送・セキュリティを矛盾なく統合すること。

複合用途は独立した一つの固定プロファイルではなく、USE-01〜10の合成として扱う。

| 管理対象 | 適用方法 |
| --- | --- |
| 建物全体 | 所有・管理体制、受変電、中央監視、防災センター、外構等の共通ルールを設定する |
| 棟・階・区画 | 実際の用途プロファイルを割り当て、清掃、入退室、作業時間、品質基準等を設定する |
| 共用設備 | 系統が供給する全用途のうち、最も大きい停止影響・安全要求を考慮する |
| 共用動線 | 来館者、居住者、従業員、搬入業者等の交差と時間帯を考慮する |
| 防火・防災 | 管理権原と用途区分を確認し、全体と各区画の計画・訓練・連絡を接続する |

## 8. 複合用途への適用例

低層階が商業施設、中高層階がオフィスの建物では、次のように分析する。

| 対象 | 適用プロファイル | 主な差分 |
| --- | --- | --- |
| 建物全体 | USE-11 | 防災センター、受変電、中央監視、共用搬入口、全体連絡網 |
| 1〜3階 | USE-02 | 長時間営業、来館者安全、店舗・厨房、営業中清掃 |
| 4〜15階 | USE-01 | テナント調整、執務時間外作業、入退館管理 |
| 厨房排気系統 | USE-02＋設備条件 | 臭気、油脂、清掃、火災予防、店舗との停止調整 |
| 受変電設備 | USE-11＋共用設備 | 商業・オフィス双方の停止影響を踏まえた停電計画 |

差分が競合する場合は、次の順に解決する。

1. 法令・安全上の必須条件
2. 人命・事業継続への影響
3. 建物全体の共通設備制約
4. 区画ごとの運用・品質要求
5. 契約上の役割と費用負担

## 9. 個別物件への適用手順

1. 建物、棟、階、区画の用途を登録する。
2. 用途マスターから基本プロファイルを選ぶ。
3. 規模、運用時間、利用者、設備、事業継続、管理方式を確認する。
4. 法令上の用途・対象設備・規模条件を別途判定する。
5. 共通業務IDごとに、適用区分と差分属性を記述する。
6. 共用設備と用途区画の依存関係、停止影響を整理する。
7. 法的義務主体、実施主体、承認者、連絡先、契約分界を確認する。
8. 現地調査、契約仕様、運用実績に基づきプロファイルを確定する。
9. 用途変更、テナント入替、設備更新、運用時間変更時に再評価する。
