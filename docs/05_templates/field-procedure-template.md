### 11.1 緊急時の原則

1. 人命と身体の安全を最優先する。
2. 危険が切迫する場合は、定められた緊急権限の範囲で停止、退避、立入制限又は通報を行う。
3. 二次災害を避け、訓練・資格・装備の範囲を超える対応を行わない。
4. 安全を確保したうえで、時刻、場所、状態、実施した措置及び影響範囲を記録する。
5. 緊急停止と復旧・利用再開は別判断とし、許可なく復旧しない。

- [ ] 定性的基準を写真例、限度見本又は観察条件で補っている

### 20.3 安全・異常対応

- [ ] 正常、要観察、異常及び危険を区別している
- [ ] 作業中止、退避、隔離及び緊急連絡の条件を具体化している
- [ ] 作業者の訓練・資格・装備を超える対応を要求していない
- [ ] 緊急停止権と復旧・利用再開権を区別している

### 20.4 役割・責任

- [ ] 実施者、現場責任者、技術・品質確認者及び承認者を区別している
- [ ] 元請け、再委託先、資格者、BM責任者、PM・FM等の役割を混同していない
- [ ] L0〜L4の判断レベルと上申条件を明記している

### 20.5 適用・証跡

- [ ] 建物用途、常駐・巡回、契約役割、法令及び責任分界の差分を確認した
- [ ] 物件固有の閾値・連絡先等を管理文書へ分離した
- [ ] 必須記録、写真、測定値、連絡、承認及び引継ぎを定義している
- [ ] 関連チェックリスト、前後手順、法令・仕様・メーカー文書への影響を確認した
