# EQC-001 受変電設備 点検・判定基準

## 0. 文書管理

| 項目 | 内容 |
|---|---|
| 基準ID | `EQC-001` |
| 対象設備 | 高圧受変電設備、変圧器、配電盤及び付帯設備 |
| 版 | 0.1 |
| 状態 | 代表例・草案 |

## 1. 対象設備と維持する機能

受電した電力を所定の電圧へ変換・配電し、保護機能を維持する。上流の受電系統、下流の重要負荷、予備電源及び接地・換気設備との関係を系統図で確認する。

## 3. 適用範囲・除外

日常の盤外目視、表示、計器、音、臭気、温度、室内環境及び承認済み非接触測定へ適用する。盤内開放、充電部への接近、保護装置試験、絶縁測定、遮断器整備、整定変更及び法定・保安規程上の資格者作業には適用しない。

## 4. 点検・判定基準

| 点検点 | 確認方法・条件 | 正常 | 要観察 | 異常 | 危険 |
|---|---|---|---|---|---|
| 受電・配電状態 | 監視表示と盤面表示を系統図・運転条件と照合 | 予定系統と一致し保護表示なし | 一時的変動又は説明可能な切替状態 | 不一致、欠相疑い、継続警報 | 焼損、発煙、火花、保護動作反復 |
| 電圧・電流・負荷 | 同一条件の基準値、前回値、相間を比較 | 適用範囲内で相互矛盾なし | 範囲内だが偏り・増加傾向 | 基準外又は急変 | 過熱・保護動作を伴う異常値 |
| 変圧器・盤外観 | 安全距離から音、臭気、変色、漏れ、表示を確認 | 通常状態で損傷なし | 軽微な汚れ、腐食又は音の変化 | 異音、異臭、変色、漏油、破損 | 発煙、火災、著しい放電音 |
| 室内環境 | 入口から浸水、結露、換気、障害物、施錠を確認 | 乾燥・換気・施錠が適正 | 温湿度・汚れ等の傾向変化 | 結露、漏水接近、換気不良、扉不良 | 盤への浸水、煙、接近不能な危険 |
| 接地・保護表示 | 外観と表示を承認範囲で確認 | 欠損・警報なし | 表示の劣化等 | 断線疑い、警報、封印異常 | 保護機能喪失が疑われ運転継続が危険 |

具体的な電圧、電流、温度、負荷率、相間差及び警報条件は、保安規程、整定表、メーカー資料及び物件別運転基準を参照する。

## 5. 操作・停止・復旧

| 行為 | 標準的な境界 |
|---|---|
| 盤面確認・記録 | 承認済み巡回範囲でL0 |
| 通常の遠隔・盤面操作 | 操作票、系統確認、資格・権限、連絡を満たす者だけが実施 |
| 保護装置の解除・整定変更 | 一般作業者は禁止。選任者・権限者の管理下 |
| 緊急停止 | 物件別緊急権限と安全に操作可能な範囲に限る。アーク・煙等がある場合は接近しない |
| 安全隔離 | 開放点、無電圧、施錠・表示等を資格者が確認 |
| 技術的復旧 | 原因、保護動作、絶縁・試験結果、下流影響を有資格者が確認 |
| 利用再開 | 技術的復旧後、重要負荷・事業影響を運用責任者が判断 |

## 6. 点検区分の境界

- 日常点検：盤外の外観、表示、計器、室内環境及び傾向確認。
- 定期点検：停止、盤内、清掃、締付、絶縁、保護継電器、遮断器等は承認済み専門手順へ引き渡す。
- 法定・保安業務：適用法令、保安規程、選任者・資格者、周期、届出及び記録を法定業務マスターで確定する。

## 7. 異常時の接続

要観察は期限付き再測定へ、異常は`PROC-EQP-005`へ、火災・発煙・放電・浸水等は接近せず`PROC-EQP-006`へ接続する。警報復帰だけで正常とせず、現場状態、保護動作理由、下流負荷及び再発可能性を確認して`PROC-EQP-007`へ引き渡す。
