入門編で使う言葉を、初学者向けに短く説明します。ここでの定義は、このガイド内で意味を揃えるためのものです。契約、法令、会社によって呼び方や範囲が異なる場合があります。

## 関係者・役割

| 用語 | 説明 |
|---|---|
| ビルオーナー | 建物の所有者。方針、投資、予算、重大リスクなどを最終判断する立場 |
| プロパティマネジメント（PM） | 個別不動産の収益、賃貸運営、予算、契約履行などを統括する機能 |
| ファシリティマネジメント（FM） | 利用組織の活動を支えるため、施設の要求や性能を管理する機能 |
| ビルメンテナンス（BM） | 建物・設備の状態と現場品質を管理し、現場業務を計画・実施・記録する機能 |
| テナント | 建物や区画を借りて利用する企業、店舗など |
| 委託者 | 業務を他者へ依頼し、契約を結ぶ側 |
| 元請け | 委託者と直接契約し、元契約の履行窓口となる受託者 |
| 再委託先 | 元請けから業務の一部を受託する事業者 |
| 協力会社 | 取引先を広く指す実務上の呼称。契約上の立場は別に確認する |
| 専門業者 | 資格、登録、許可、メーカー技術などが必要な専門業務を担う事業者 |

## 作業・状態

| 用語 | 説明 |
|---|---|
| 運転 | 設備を動かし、状態を監視・調整すること |
| 監視 | 計器、警報、画面などから設備や建物の状態を継続的に見ること |
| 巡回 | 建物内を回り、目視、音、臭い、計器などから状態を確認すること |
| 検針 | メーターの値を読み取り、使用量や状態を記録すること |
| 点検 | 対象の状態を確認し、正常か、異常があるかを判断すること |
| 保守 | 良好な状態を保つために清掃、調整、注油、部品交換などを行うこと |
| 修繕 | 不具合や劣化を直し、必要な状態へ戻すこと |
| 工事 | 設備や建物へまとまった変更を加える作業。修繕との境界は条件により異なる |
| 復旧 | 異常や停止の後、技術的に必要な状態へ戻すこと |
| 利用再開 | 安全や利用への影響を確認し、停止していた設備・区域を再び使えると判断すること |
| 応急処置 | 本格修繕までの間、危険や影響の拡大を抑える暫定的な対応 |
| 是正 | 基準に合わない状態や仕事の結果を直すこと |

## 確認・記録

| 用語 | 説明 |
|---|---|
| 検査 | 技術・品質などの基準を満たすか確かめること |
| 自主検査 | 作業を行った側が、提出前に自ら結果を確認すること |
| 検収 | 発注者などが、契約上求めた成果として受け取れるか確認すること |
| 承認 | 権限を持つ人が、内容や次の行動を認めること。何を認めたかを区別する必要がある |
| 作業記録 | いつ、誰が、何を行い、どのような結果だったかを残す記録 |
| 証跡 | 後から実施や判断を確かめるための記録、写真、測定値、受領記録など |
| 台帳 | 建物、設備、鍵、図面、履歴などを継続的に管理する基準情報 |
| 速報 | 詳細が未確定でも、安全や初動のために早く伝える報告 |
| 確報 | 事実、原因、影響、対応などを確認後にまとめる正式な報告 |

## 契約・法令

| 用語 | 説明 |
|---|---|
| 仕様 | 業務の対象、方法、頻度、品質、提出物など、求める内容を定めたもの |
| 法定業務 | 法令に基づき、対象、周期、資格、報告、保存などの条件が定められる業務 |
| 法的義務主体 | 法令上、実施、選任、届出、報告、保存などの義務を負う人または組織 |
| 管理方式 | 常駐、巡回、遠隔監視など、いつ・どこから・どの体制で管理するかという方式 |
| 常駐管理 | 担当者が対象建物に継続的に配置される管理方式 |
| 巡回管理 | 担当者が複数物件などを回って管理する方式 |
| 遠隔監視 | 建物外から設備の状態や警報を監視する方式 |

詳しい関係は[ビルメンテナンスとは](../what-is-building-maintenance/)と[関係者と役割](../people-and-roles/)を参照してください。この用語集は後続の本文・リファレンス整備に合わせて拡充します。
